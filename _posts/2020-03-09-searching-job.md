## 1.简历

简历的重要性，甚至超过你的颜值，认真对待的关键：要知道怎么写简历

 1. 锁定求职意向
- [x] 从用人单位招聘广告里匹配
- [x] 简历里一定要重点显示自己的求职意向

 2. 细心匹配素材
- [x] 头脑风暴联想法：
将与求职相关的有效信息，例如 接受教育经历；工作经历；社团经历；奖励证书等 写进简历。
- [x] 技能类：
匹配工作的专业技能；可迁移能力 例如沟通组织 语言表达能力；自我管理技能 例如吃苦耐劳bulabula~
- [x] 关键词类：
从招聘简章里画出重点，为最后一步做准备。
- [x] 断舍离：
把简历内容浓缩进一张纸当作目标，这是常见的一个采坑点。

 3. 进一步完善
==突出你的专业度==
- [x] 将事例中的数据转换为**百分数**等非文字形式
- [x] 避免出现**错别字**等低级毛病
- [x] 根据招聘单位**具体要求**选择投递简历的方式

## 2.网申

顾名思义，是一种网上求职的招聘形式。

 1. 开放式问题
- [x] 成就故事
- [x] 实习经历
- [x] 为什么选我
- [x] 职业目标
==并不是随便发挥，而是找到落脚点，回归求职这个本质问题上来。==

 2. 素材参考
- [x] 招聘广告
- [x] 公司官网
- [x] 企业年报
- [x] 企业文化
==取词关键，落笔实处。==

 3. 多做有益的事
 **例如，多了解一些同行业领域内不同企业的资料。**
 
 4. 网申过程易踩的坑
- [x] 低级错误易踩雷区：
错别字，英文不规范等
- [x] 准备充分自己的“面经”：
就象我这样把面试求职当作一门功课去学，别问我为什么（骚年，走上社会时你体会到的学习：在学校的学习才是新手级）
- [x] 时刻备份难得的灵感：
与计算机和网络打交道，要懂得**一保存 二备份 三检查**。

## 3.面试

走到这里，基本功就告一段落了。

 1. 求职常规流程图
 
![求职流程图](./attachments/1583723808568.drawio.html)
- [x] 上机考试：
一般只有较专业职位会有这一项，例如计算机相关
- [x] 集体面试：
主要有**自我介绍 提问 讨论**等环节
- [x] 业务面试：
主要包括**自我介绍 考察专业知识**
- [x] 综合测评：
顾名思义了
- [x] 综合面试：
包括**自我介绍 交谈（谈人生理想，向面试官问一些公司相关话题）**

==总之，各个面试环节侧重点都有所不同，自信使然就好。==

 2. 面试技巧
 根据经验，面试多考察应聘者以下几方面：
- [x] 辩论能力：
也就是逻辑思维能力
- [x] 压力测试：
多考察耐压能力
- [x] 诚信：
不要一心只为求职，有一说一把握分寸即可
- [x] 是非观：
阿弥陀佛

 3. 自我介绍
- [x] 大方向把握：
一定要表现**真实**的自己；
**会讲故事**；
**有节奏感**
- [x] 具体方法论：
**个性化**介绍自己；
讲故事四要素：**情景**+**任务**+**行动**+**结果**
*（1.情景：为了完成课题组老师给出的试验任务，要在300份样品中挑选出合适的样品；2.任务：此次任务由我们小组5人，我任组长，要在3天时间内完成；3.行动：首先我本着方法论原则，利用半天时间学习了以往相似实验的基本操作流程，设计实验流程和解决方案，以及如何利用统计学工具，第二天我们小组分工合作，女同学负责需要认真细致的统计工作，男同学处理样品，最后的实验结果有小组一同审核；4.结果：第三天，我们将结果及具体操作流程详述给课题组老师，出色的完成了此次试验任务。）*
- [x] 进一步完美：所述内容与工作**匹配**，展现对工作的**热情**，**主次分明 详略得当**，自信中肯<i class="fab fa-angellist" style="color: #FF9800"></i>

 4. 无领导小组讨论
- [x] 团队合作：
出结果不重要，难得的是从小组讨论中确定自己的角色
- [x] 小组讨论四大分工：
掌控**时间**，提出**建议**，有效**记录**，适当**发言**

 5. 一对一面试
 **本节内容较多，单独成章。**
 
 ## 4.一对一面试

 1. 基本方法论
- [x] 夸公司职位：
优势，前景等
- [x] 匹配自己：
宣传自己，推销自己
- [x] 表表决心：
诚恳求职

 2. 应变原理
- [x] 如何向面试官提问：
匹配和**落脚点**都应该是职位或者公司
- [x] 应对新奇问题：
落脚点依然是从职出发

3. 面试礼仪
- [x] 穿/动/说：
着装整洁，举止得体，谈吐礼貌
- [x] 从企业角度思考问题

4. 求职三岔口
- [x] 外企/民企：
求职的两大方向
- [x] 公务员/国企：
俗称“铁饭碗”
- [x] 学校/科研院所：
多为继续深造，向成为大牛努力
