## Abstract

- [ ] 猪智齿病毒(PSV)已在世界范围内被检测到猪群中。虽然PSV会弓|起各种症状，如猪的脑脊髓炎、腹泻和肺炎，但PSV感染的经济影响仍有待确定。
- [ ] 然而，关于PSV的分布和遗传多样性的资料相当有限，特别是在非洲。
- [ ] 在本研究中，我们调查了赞比亚PSV的流行情况，并对分离的PSVS进行了遗传学和生物学特性的研究。
- [ ] 对201 8年采集的147份粪便标本进行了筛查，发现乳猪和育肥猪的PSV感染率较高(分别为36.2%和94.0%)。
- [ ] 系统发育分析表明，赞比亚PSV在由中国菌株组成的类群中被划分为三个不同的谱系(1~3系)。在Vero E6和BHK细胞中,属于第2和第3血统的赞比亚PSV比于第1血统的PSV的复制效率更高。
- [ ] 生物信息学分析表明，发生了遗传重组事件，重组断点位于L和2A基因上。
- [ ] 结果表明，至少有两种不同的PSV在赞比亚猪群体中存在，遗传重组在猪群的进化中起着重要的作用。

## 1. Introduction

- [ ] Sapelovirus屢，瞩皮氏科，由两种组成:Sapelovirus A和Sapelovirus B,包括猪脑病毒(PSV)和猴脑病毒(Sapelovirus A,Sapelovirus B)
- [ ] PSV管被命名为猜肠病毒8型PEV-8).PSV是一种无包膜、阳性、单链RNA病毒 ,其基因组大小约为7.5kb,由50个非翻译区(UTR)、较大的开放阅读框(ORF)、30个UTR和多腺嘌呤尾(PolyA)组成。
- [ ] ORF编码由一个领导蛋白(L)、4个结构蛋白(VP 4、VP2、 VP3和VP1)和7个非结构蛋白(2A、 2B、 2C、 3A、 3B、 3C和3D)组成的单个多蛋白。
- [ ] VP1、 VP2和VP3是主要的衣壳蛋白，位于病毒表面，VP1序列是PSV编码蛋白中最多样的。
- [ ] 自1958年在英国首次发现PSV以来，在许多国家都发现了PSV,包括美国、巴西、澳大利亚、日本、中国、韩国、印度、德国、法国、意大利、捷克共和国和英国。
- [ ] 在肯尼亚和乌干达，猪粪便中检测到PSV的基因组片段，但非洲PSV的流行率和遗传多样性尚不清楚。
- [ ] 猪的PSV感染在田间条件下往往是亚临床的。在亚洲和欧洲，腹泻猪群中的PSV感染率很高，而在无症状的猪群中也经常检测到PSV.
- [ ] 事实上，PSV会在实验感染的猪体内引起各种症状，例如脑脊髓炎、腹泻和肺炎。
- [ ] 因此，美国、英国和中国最近报道了-些与PSV感染引|起的呼吸道症状相关的神经病和胃肠炎的爆发。
- [ ] 发病猪表现为腹泻、呼吸窘迫、共济失调、偏瘫、精神迟钝，对环境刺激反应减弱，临床发病后数天内死亡。虽然已报道了PSVS的遗传多样性，但由于遗传特征，PSV菌株之间的表型差异尚未得到讨论。
- [ ] 目前对猪PSV感染的发病机制知之甚少，因此PSV的存在与临床症状之间的病因关系仍存在争议。
- [ ] 在本研究中，我们调查了赞比亚猪群中PSV感染的流行情况，并测定了从粪便样本中分离出的PSV的全基因组序列。对分离的PSV进行了遗传和生物学鉴定。

## 2. Materials and Methods

### 2.1. Sample Collection

**样本收集**
- [ ] 我们重点关注拥有500多头猪的农场，并选定了5个农场，所有者同意在这些农场进行抽样(卢萨卡农场A和E、奇兰加农场B、卡夫埃农场C和奇博博区D农场)，并在2018年1月至12月期间向当地兽医人员采集了样本(表1)。
- [ ] 为了收集粪便样本，随机选择了- -些笔,并将地板上的新鲜粪便收集成管子。取哺乳(0~ 3周龄)和育肥猪(4~ 12周龄)有腹泻或无腹泻的粪便标本。

### 2.2. Detection of PSV by RT-PCR

**RT-PCR检测PSV**
- [ ] 粪便样品悬浮在含5%抗生素抗真菌溶液的磷酸盐缓冲溶液(PBS)中，并进行了短暂离心。用TRlzol-LS(ThermoFisher Science ,Waltham,美国)根据制造商的指示从上清液中提取总RNA.
- [ ] 用RT-PCR和PrimeScript-步RT-PCR试剂盒,根据50UTR内的保守序列设计特异性引物，对RNA样品进行PSV基因组检测。2(Takara, Shiga, 日本)根据制造商的协议(补充表S1)。
- [ ] RT-PCR条件为:在50-C下合成30 min,94C激活2 min, 94C激活30s，53-C 30s，72=C 40s, 72°C延伸7 min.
- [ ] 用琼脂糖凝胶电泳分离PCR产物，用溴化乙锭染色在紫外线照射下观察。

### 2.3. Cells and Virus Isolation

**细胞与病毒分离**
- [ ] 本研究以非洲绿猴肾细胞(Vero E6)和小仓鼠肾细胞(BHK)为研究对象。
- [ ] 这些细胞保存在Dulbecco改良的Eagle 's培养基(D-MEM)中，添加10%胎牛血清(FBS)、2mml-谷氨酰胺100单位/mL青霉素、100μg/mL链霉素、3.5mg/mL D-葡萄糖和1.0mg/mL NaHCO 3(37=C+5%CO2).
- [ ] 将粪便悬液上清液接种到Vero E6和BHK细胞中，37°C与5%CO2孵育1小时进行病毒吸附。
- [ ] 接种后，用PBS洗涤两次，在含2%FBS、2mm-Glu、4%抗炎剂和1.0 mg/mLNaHCO 3(37°C, 5%CO2)的Eagle 's最低必需培养基(日本东京奈西药业有限公司)中进行保存。
- [ ] 将Vero E6和BHK细胞传代3次，观察细胞病变效应(CPE)。体外培养的CPE细胞经0.45m孔膜滤池(日本东京lwaki)过滤去除细菌细胞，再接种到相应的细胞株中。
- [ ] 用RT-PCR和下一代测序证实了- -株PSV的分离.我们进一步证实, 通过下-代测序，上清液中没有其他病毒。

### 2.4. Whole Genome Sequencing

**全基因组测序**
- [ ] 培养的培养基从显示CPE的细胞中分离出来,并短暂离心去除细胞碎片。
- [ ] 用QIAamp病毒RNA迷你试剂盒(Chiagen,Hilden,德国)从上清液中提取总RNA,并根据制造商的指示用PrimeScript双链cDNA合成试剂盒(Takara, Shiga, Japan)合成cDNA。
- [ ] 双链cDNA片段和索弓|适配器被附加使用NextEra XT DNA库准备工具包llumina,圣地亚哥，CA,美国)根据制造商的指示。
- [ ] 利用MiSeq试剂Kit V3(600个周期)在MiSeq仪器上对文库进行全基因组测(lumina,San Diego, CA, USA).序列数据用CLC基因组学工作台软件(CLC BIO, Chiagen,Hilden,德国)进行分析。
- [ ] 在对低质量读取进行修整后，其余的读取将使用默认设置重新组装利用从NCBI数据库下载的病毒参考资料进行局部BLAST分析，确定病毒种类。
- [ ] 通过映射到PSV参考序列获得的一致序列(GenBank登录号。KJ 463384和KX574284)与新组装的PSV连体一致。
- [ ] 最后，将所有修剪后的读序列重新映射到对齐的PSV序列上，得到覆盖超过20条的一致序列。
- [ ] 此外，采用RT-PCR和PSV特异性弓|物快速扩增cDNA末端(RACE)方法，对全基因组的50个和30个末端区域和VP1 C-端区进行扩增,用Sanger测序法(补充表S 1)确定低覆盖区的序列。用PrimeScript一步法RT-PCR试剂盒进行RT-PCR和RACE。2和更智能的RACE cDNA扩增试剂盒(Takara，志贺，日本)。
- [ ] 从琼脂糖凝胶中纯化PCR产物，用BigDye终止子v3.1循环测序试剂盒(生命技术，应用生物系统，福斯特市, CA,美国)和ABI PRISM3130(生命技术，应用生物系统，福斯特市，CA,美国)进行直接测序。

### 2.5. Phylogenetic Analysis and Other Bioinformatic Analyses

**系统发育分析和其他生物信息学分析**
- [ ] 通过与已知PSV株基因组的比较，预测了PSV的完整ORF和裂解位点。对从DDBJ/EMBL-Bank/GenBank数据库获得的各种PSV序列进行生物信息学分析(补充表S2)。
- [ ] 利用PhyML3.0Web服务器和Akaike信息准则，对完整的ORF、VP1和3C+ 3D(3CD)核苷酸序列进行系统发育分析.
- [ ] 利用肌肉软件对序列进行比对，利用GTR+G+I模型和1000个自举复制的最大似然方法构建系统发生树。
- [ ] 利用肌肉对VP1 C端区和多蛋白的氨基酸序列进行排列。使用GENETYX版本1 2在赞比亚和其他具有代表性的PSV之间进行了全球身份分析(GENETYX公司，日本东京)。
- [ ] 重组分析使用Simplot软件版本3.5.1和重组检测程序(RDP)版本4.97进行

### 2.6. Growth Kinetics of PSV

**PSV生长动力学**
- [ ] 用Vero E6和BHK细胞分离的PSV分别在VeroE6和BHK细胞中传代。用Vero E6细胞检测树突状病毒的空斑形成单位(PFU)。
- [ ] 用BHK细胞测定50%的组织培养感染剂量(TCID 50),因为在感染PSV的BHK细胞中未见明显的斑块形成。在菌斑分析中，Vero E6细胞汇合的单层细胞被10倍系列稀释病毒感染。
- [ ] 吸附1h后，用含2%FBS、2mm L-谷氨酰胺、100单位/mL青霉素、100 g/mL链霉素、1.0mg/mL NaHCO 3和1 %琼脂的MEM 覆盖细胞单层。感染后2天测定菌斑形成单位。TCID 50每稀释4井, BHK细胞汇合单分子被10倍串联稀释病毒感染。
- [ ] 感染后3天观察cpe,用red-muench法计算病毒滴度。
- [ ] Vero E6和BHK细胞分别在感染倍数(MOI)为0.01的情况下感染PSV。
- [ ] 吸附1h后，去除接种物，在含5%CO2的37oC条件下，分别与含2%FBS、2mmL-谷氨酰胺、100单位/mL青霉素、100 g/mL链要素和1.0mg/mL NaHCO 3的MEM孵育。
- [ ] 分别于感染后8、24、 48和72h收集感染Vero E6细胞的培养基，感染后8、24、48、72和96h分别收获感染BHK细胞的培养基。
- [ ] 3000 rpm, 4-C离心5 min, 去除细胞碎片。用Vero E6和BHK细胞分别用空斑法和TCID50法测定培养上清液的病毒滴度。

### 2.7. Statistical Analysis

**统计分析**
- [ ] 采用卡方检验分析PSV感染率差异，P值<0.05被认为具有统计学意义。

## 3. Results

### 3.1. Prevalence of PSV Infection in Zambia

**赞比亚PSV感染流行情况**
- [ ] 2018年，分别从赞比亚的乳猪和育肥猪中采集了47份和100份粪便样本。在这些样本中，40份和59份分别来自腹泻乳猪和育肥猪(表2)。
- [ ] 从这些样品中提取RNA，用特异性引物RTP-CR检测PSV基因组的50 UTR(补充表S1)。
- [ ] 47份乳猪样品中有17份(36.2%)检出PSV基因组。在RT-PCR阳性标本中，健康仔猪和腹泻母猪分别为42. 9%/3/7)和35.0%(14/0(02和补充表S3)。
- [ ] 健康仔猪与腹泻乳猪的阳性率差异无统计学意义(P=0.69)。育肥猪PSV检出率极高，100份标本中94份(94.0%)PSV基因组阳性。
- [ ] 健康标本和腹泻标本中病毒基因组检出率分别为95.1%(39/41)和93.29(55/59)(表2和补充表S3).健康和腹泻型育肥猪的患病率差异无显着性(P=0.69)。
- [ ] 值得注意的是，育肥猪的PSV感染率(94.0%)显著高于乳猪(36.2%)(P<0.01)。

### 3.2. Isolation and Full Genome Sequence of PSVs

**PSVS的分高及全基因组序列**
- [ ] 为分离病毒，将111份PSV RT-PCR阳性粪样接种于Vero E6和BHK细胞中，接种细胞内可见明显的CPE(图1，补充图S1)。
- [ ] 最后，分别从Vero E6和BHK-21细胞中分离到8株和25株分离株，表明BHK细胞对PSV感染更敏感。
- [ ] 6株分离物(PSV-20、PSV-21，PSV-22.、PSV-23、PSV-26和PSV-46)根据 系统发生部位(见下文)作为代表进行进一步分析(表3)。
- [ ] PSV-20只从Vero E6细胞(PSV-20-V株)中分离，而PSV-22和PSV-26只从BHK细胞中分离(分别为PSV-22-B和PSV-26-B株)。
- [ ] 从这两个细胞系中分离出PSV-21-V和PSV-21-B、PSV-23(PSV-23-V和PSV-23-B)和PSV-46-V(PSV-46-V和PSV-46-B)。
- [ ] 为了获得病毒的全基因组序列，从每个分离物提取的总RNA中提取的DNA文库进行了下代测序。通过生物信息学分析，获得了全长7000多个核苷酸的PSV基因组序列。
- [ ] 对RT-PCR和RACE-PCR的扩增产物进行Sanger测序，以确定低覆盖区(即50和30末端和VP1 C-末端区域)的序列。最后，我们获得了由7458-7498个核苷酸组成的完整的病毒基因组序列(表3)。
- [ ] 赞比亚PSVS的全基因组序列以DDBJ/EMBL/GenBank LC 508226-LC508234为登录号，存放在日本DNA数据库(DDBJ)中。

### 3.3. Comparative Genome Analysis among PSVs

**PSVS的比较基因组分析**
- [ ] 通过与其他已知PSV的比较分析，预测了PSV分离株的全ORF序列。并与V13株的氨基酸序列(GenBank登录号)进行比对，推导出相应的氨基酸序列。(补充表S4)
- [ ] 4个分离物(PSV-21、PSV-22、 PSV-23和PSV-46)的ORF大小为6969个核苷酸.
- [ ] 4个分离物(PSV-21、PSV-22、PSV-23和PSV-46)的ORF大小为6969个核苷酸.PSV-20和PSV-26分别有较长的ORF、6996和6993个核苷酸(表3)。
- [ ] 对赞比亚和其他已知PSVS的比对表明，PSV-20和PSV-26分别在VP1的C-端插入了8个和9个氨基酸(补充图S2)。
- [ ] 这些插入在中国、韩国、日本、美国、法国和德国分离的其他菌株中也存在，在- -些PSV菌株中也很常见。
- [ ] 与其它大陆分离的PSV株比较，赞比亚PSVS核苷酸序列和氨基酸序列的同源性分别为83.3%~90.69%和92.6%~98.5%(补充表S5)。
- [ ] 所有6种赞比亚PSV与中国菌株的亲缘关系最为密切。
- [ ] 在赞比亚PSVS中，核苷酸序列和氨基酸序列的同源性分别为85.6%~99.5%和93.7%~99.7%。1月份在卢萨卡同-农场分离到PSV-20、PSV-21、 PSV-22和PSV-23，而PSV-20的核苷酸序列(89.5%~89.6%)和氨基酸序列(95.1%~95.39%)与其它3个分离株的同源性较低。
- [ ] PSV-20与在奇兰加另一个农场发现的PSV-26具有较高的同一性.这些数据表明，两个以上的遗传不同的PSV同时在一个农场流通。
- [ ] 此外，还比较了赞比亚PSVS(补充表S6)中病毒蛋白序列的同源性。
- [ ] VP1是一种衣壳蛋白，在12种病毒蛋白中差异最高，氨基酸序列同源性在78.9%~99.6%之间。
- [ ] 而2B、2C、3A、3C、3D等非结构蛋白具有较好的保守性，氨基酸同源性范围为97.0%~100%。

### 3.4. Phylogenetic Analyses of PSVs

**PSVS的系统发育分析**
- [ ] 利用赞比亚和其他有代表性的PSV的序列数据，从GenBank检索到-棵基于完整ORF核苷酸序列的系统发育树(图2a,补充表S2)。
- [ ] PSVS在系统发育上分为三个类群(Clade A到C)。CLADE A只包括中国菌株。CLADE B包括在日本美国和韩国发现的菌株。CLADE C是由英国和印度菌株组成的。
- [ ] 两个欧洲菌株不属于这些类群。
- [ ] 将所有赞比亚PSV分为Clade A,赞比亚PSV又分为3个不同的谱系，1系(PSV-21, PSV-22，PSV-23), 2系(PSV-20和PSV-26), 3系(PSV-46)。
- [ ] 对VP1和3CD核苷酸序列进行了系统发育分析。在基于VP1基因的系统发育树中，赞比亚PSVS被划分为三个谱系，即1、2和3，其方.式与完全ORF树相同(图2b)。
- [ ] PSV-20和PSV-26分别与Jpsv 447和Jpsv1315等Clade B毒株聚在一起，PSV-46形成-个单独的分支。以3CD为基础的树表明，赞比亚PSV是系统遗传分离成两个不同的类群，并与Clade A菌株形成一个单系类群。
- [ ] PSV-20与1系病毒比2系(即PSV-26)病毒关系更密切(图2c)。
- [ ] 这些结果表明，PSV-20、 PSV-26和PSV-46在不同遗传类型的PSV之间发生了遗传重组

### 3.5. Recombination Analyses of PSVs

**PSVS的重组分析**
- [ ] 为了获得赞比亚PSVS基因重组的证据，利用Simplot软件和RDP进行了重组分析。

## 4. Discussion

- [ ] 虽然PSV是一种常见于欧洲、 亚洲、澳大利亚和北美至南美洲的猪种，但在非洲，PSV感染的流行率和地理分布以及遗传多样性是完全不清楚的。
- [ ] 在这项研究中，我们证明了PSV感染在赞比亚猪群中是普遍存在的。在我们的筛选中，育肥猪的PSV基因组流行率(94.0%)高于喃乳猪(36.2%)(表2)。
- [ ] 在捷克和中国的猪群中也观察到了这种情况。这可能是由于母猪抗体下降所致，猪的易感性较高。
- [ ] 另外，猪也可能通过持续感染成为PSV携芾者。然而，由于PSV感染的长期临床过程(包括脱毒时间)尚不清楚，因此很难台理解释育肥猪PSV的高发病率。
- [ ] PSV按系统演化分为与地理位置相关的不同的聚类。
- [ ] 基于PSVS全基因组序列的系统发育分析表明，赞比亚PSVS在Clade A中形成了三个不同的谱系，即1、2和3(图2a)。
- [ ] 由于在本报告之前，这类病毒只有中国菌株，可以想象赞比亚的PSV起源于中国菌株。
- [ ] 虽然赞比亚农民一般在国内进行猪的贸易,很少通过国际贸易引进新的猪，但中国流行的PSV有可能伴随着国际人类运动，导致中国PSV传入赞比亚。
- [ ] 事实上，有报道称，在国际旅客从中国带来的肉类产品中，发现了高致病性离流感病毒和非洲猪瘟病毒等国际兽疫病原体。
- [ ] 然而，PSV的遗传数据过于有限，不能对PSV的全球传播模式和进化历史给予合理的解释。
- [ ] 需要增加流行病学和序列数据，以更好地了解PSVS的演变。
- [ ] 般来说，重组、缺失、插入和点突变是非节段病毒遗传和表型进化的主要模式。最近，在日本和中国，已有证据表明PSVS之间存在重组，可能的重组断点在2B基因的上游区域。
- [ ] 本研究在赞比亚分离到一些基因重组病毒,包括PSV-20、PSV-26和PSV-46.系统发育分析表明，PSV-20和PSV-26的衣壳蛋白来源于相同的Clade B祖先，而它们的非结构基因是从两个不同的Clade A菌株中获得的。
- [ ] 另一方面，PSV-26和PSV-46的衣壳蛋白可能来自不同的亲本品系，而它们的非结构基因很可能来源于同一株祖先的Clade A株(图2)。重组分析预测PSV-20和PSV-26最可能的重组断点位于L和2A区域，但无法确定PSV-46的断点(图3)。
- [ ] 这些结果表明赞比亚PSV的遗传多样性是由几个重组事件演化而来的。虽然pSV被认为对猪非致病性或低致病性，但最近在多个国家报告了由psv感染弓|起的各种严重疾病的爆发。
- [ ] 因此，PSVS可能通过基因重组获得较高的致病性。
- [ ] 我们发现，谱系2和3病毒的体外生长效率高于谱系1病毒(图4)。1系病毒特有的大部分氨基酸残基存在于衣壳蛋白中，即VP1、VP2、 VP3和VP 4(84个氨基酸残基中的8 1个)(补充表S8)。
- [ ] 这些衣壳蛋白共同介导病毒生命周期的基本步骤，如进入病毒基因组进入受感染的细胞，以及病毒的组装和释放。
- [ ] 因此，可以合理地假设这些衣壳蛋白是有效的病毒体外复制和潜在的病毒致病性。
- [ ] PSV-20具有2-衣壳蛋白谱系1-骨干，体外生长能力较强，这也支持了这一假说。
- [ ] 猪场自然感染PSV是否弓|起腹泻等疾病仍有争议。
- [ ] 在本研究中，在腹泻猪和非腹泻猪中都检测到了PSVS(表2).
- [ ] 事实上，经常有报道说在没有症状的猪身上发现了PSV.然而，实验性的PSV感染无疑会弓|起猪的腹泻、脑脊髓炎和肺炎。
- [ ] 事实上，在美国、英国及中国，均有3至12周龄猪患神经病、肠胃炎及呼吸系统疾病的致命感染个案。猪圆环病毒-2在世界猪群中也茸適存在
- [ ] 虽然猪圆环病毒-2感染本身是无症状的，但受感染的猪偶尔会在病毒混台感染或兔疫刺激时出现严重的断奶后多系统衰竭综合征。同样，PSV也可能需要不明因素来显示猪的全部致病性。
- [ ] 为了正确估计PSV感染对养猪业的负面影响，需要进-步的研究，以更好地了解PSV感染的流行病学及其致病性的分子机制。
