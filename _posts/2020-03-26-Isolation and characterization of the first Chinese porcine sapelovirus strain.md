## Abstract

- [ ] 对上海某猪场猪急性腹泻、呼吸窘迫和脊髓灰质炎的病因进行了调查。从受感染动物的样本中检测出已知在猪身上引起类似症状的病毒和细菌，只分离到猪的皂甙病毒(PSV;命名为CSH株)。
- [ ] 在敏感细胞中观察到的特异性细胞病变效应以及PCR和电镜结果进一步证实了PSV的存在。核苷酸测序和系统发育分析表明该分离物为PSV.
- [ ] 当接种到健康猪体内时，PSV.csh弓 |起的症状与在受影响的猪群中观察到的症状相同。因此，psv.csh是本病的致病因素。
- [ ] 据我们所知，这是我国首次报道PSV感染仔猪。

## Introduction

- [ ] Sapelovirus(来自猿猴、离和猪肠类病毒)是皮氏科的一个新属。目前，该属包括猪脑病毒、猿猴皂甙病毒和离流感病毒三种。
- [ ] 与毕赤病毒科的所有成员相似，这些病毒含有7.5-8.3kb的单链、多腺嘌呤、正感基因组RNA.在50-端有一个基因组连接蛋白(Vpg), 还有一个30-polya尾。
- [ ] RNA编码一个单- -的多蛋白，随后被加工成功能性产品。主要的结构蛋白编码在N-端附近。猪智囊病毒(PSV)，正式命名为猪肠道病毒8型(PEV-8)，被归类为猪肠道病毒A(PEVA)。
- [ ] 然而，进-步详细的遗传和系统发育分析表明，PEV-8具有独特的基因结构，包括与其他猪肠病毒具有遗传区别的L和2A基因区域。
- [ ] 因此，它被归入一个新的属，Sapelovirus,连同几种密切相关的鸟类和猴皮病毒。
- [ ] 与猪病毒(Ptv)和猪肠病毒相比，ptv病毒和猪肠病毒在猪肾细胞中具有独特的I型细胞病变效应(Cpe)。
- [ ] 与PTV和PEV相似，PSV是导致严重和轻度神经疾病、生殖功能衰竭、肺炎和腹泻的致病因素。已从PSV感染中恢复的猪仍然排出病毒，是重要的感染源。
- [ ] 目前，PSV仅由一种血清型组成，pSV感染由PCR诊断，取代了细胞培养中的病毒分离和血清学检测对血清型的分化。
- [ ] 在我国，猪圆环病毒(PCV 2)和猪繁殖与呼吸综合征病毒(PRRSV)是目前弓|起猪呼吸道疾病和脊髓灰质炎最重要的病原体。然而，据我们所知，还没有关于猪PSV感染和分离的研究报告。
- [ ] 在目前的研究中，我们报告了中国首次爆发与呼吸道疾病和脊髓灰质炎相关的PSV诱导的胃肠炎。

## Materials and methods

### 1.Background and samples

**背景和样本**
- [ ] 与呼吸道疾病和脊髓灰质炎有关的胃肠炎爆发于2009年12月，发生在位于中国东南部上海郊区的一个小型商业猪场。附近两个养殖场的17头仔猪出现腹泻、呼吸窘迫、共济失调、腿麻痹和死亡等症状。
- [ ] 两个饲养场的仔猪分别为50日龄和60日龄。采集17头仔猪粪便、血清、肝脏、脾脏、肺、肠、脑标本。
- [ ] 为了避兔样本污染，从动物肛门直接获取粪便标本，取样时使用一-次性材料。

### 2.Pathogen detection

**病原检测**
- [ ] 进行了病毒和细菌检测的诊断程序，仅检测到了皂甙病毒。

### 3.Virus detection

**病毒检测**
- [ ] 用TRIzol试剂(中国塔卡拉)根据制造商的指示从粪便标本和器官中提取总RNA.
- [ ] 在含14μRNA、150 ng反向引物、20mm/dNTP、40 U RNasin、200 U小鼠白血病病毒逆转录酶(M-MLVRT，Takara, 中国)的反应混合物中台成了cDNA,总体积为20μl,在42.0C下孵育60 min.
- [ ] 基因组DNA是用先前报道的方法从标本中提取的。
- [ ] 应用DNA/RNA进行PCR或逆转录PCR(RT-PCR)检测猪轮状病毒(PoRV-A、PoRV-C).猪肠道病毒B(PEV B)、PSV. PTV.猪胼胝病毒、猪星形细胞病毒(PAstV)、猪细小病毒(PPV)、PCV 2、PRRSV. 伪狂犬病病毒(PRV)、典型猪瘟病毒(CSFV)、 猪传染性胃肠炎病毒(TGEV)、猪流行性腹泻病毒(PEDV)和乙型脑炎病毒(JV)。
- [ ] 用1.5%琼脂糖凝胶电泳分离扩增产物，用AxyPrepDNA凝胶提取试剂盒(美国Axygene)进行纯化。
- [ ] 将纯化的PCR产物克隆到PMD-18T载体(Takara,中国)中，进行测序(大连保生武)。例是使用BLAST搜索程序识别的。

### 4.Bacteria detection

**细菌检测**
- [ ] 标本中的细菌采用标准的好氧和厌氧细菌培养方法培养，试验程序采用预还原哥伦比亚琼脂和MacConkey 's琼脂。培养96 h后，传代培养不同菌落类型，并进行表型和生化鉴定。
- [ ] 根据这些特性无法识别的细菌被进行了聚合酶链反应(PCR)。用PCR法检测支原体、钩端螺旋体、链球菌、李斯特菌和布鲁氏菌的存在。

### 5.Virus isolation

**病毒分离**
- [ ] 从PCR阳性粪样中分离病毒。在无菌磷酸盐缓冲液(pbs，pH7.2-7.4)中， 将阳性粪便转化为10%(w/v)悬浮液，上清液经0.2 Im注射器过滤。
- [ ] 随后，将过滤后的悬浮液应用于pK-15(猪原代肾)细胞，37°C作用1h.
- [ ] 然后丢弃接种物，加入添加1%胎牛血清的最小必需培养基(MEM)。PK-15细胞有传至CPE出现。
- [ ] 然后对具有不同CPE的培养物进行三次附加传代，并测定其滴度(TCID 50)。分离的病毒株命名为PSV.csh.

### 6.Electron microscopy

- [ ] 分离株pK-15细胞经2000 9g离心10min(Eppendorf 5804R)后离心3h(贝克曼SW 41)。
- [ ] 产性的颗粒被重新悬浮在少量的PBS中，并被雾化在涂层电子显微镜(EM])网格上。
- [ ] 用1%磷钨酸(pH7.0)染色，透射电镜(JEOL-JSM-1200)观察病毒的形态鉴定。

### 7.RT-PCR detection and nucleotide sequence analysis

**RT-PCR检测及核苷酸序列分析**
- [ ] 为了确定EM在细胞培养上清液(第八代)以及自然感染仔猪的原始病料中观察到的病毒是否为PSV，采用PSV特异性引物进行RT-PCR，如前所述
- [ ] 梘据制造高的指示，用TRIzol试剂从感染病毒株的pK-15细胞培养物中提取总RNA。用小鼠白血病病毒逆转录酶合成cDNA。
- [ ] 然后，用设计的弓|物识别PSV 50-UTR内高度保守的序列，用于cDNA的PCR检测。扩增产物经电泳分离并测序。
- [ ] 序例是使用BLAST搜索程序识别的。
- [ ] 为了进-步鉴定PSV.csh菌株， 还测定了该菌株的全序列。
- [ ] 根据已发表的PSV-V13株基因组序列(GenBank ID: AF 406813),设计了10个引物组，以扩增PSV.csh完整ORF序列的重叠区。
- [ ] 分离物PSV.csh的50-UTR和30-UTR序列是根据制造商的指示，使用50-30种族试剂盒(Takara,中国)获得的.该序列已提交GenBank(GenBankID: HQ875059)。
- [ ] 利用MegAlign软件(DNAStar InC.,麦迪逊，WI, USA)对PSV.csh的基因组序列进行分析，然后用CLUSTAL X v1.82程序(USA)将其与已发表的Picor导航科其他成员的基因组序列进行比较。
- [ ] 利用MEGA 4.1软件(USA),采用邻接法建立了P1基因区的系统发育树，并利用MEGA4.1软件进行了启发式搜索，获得了100个自举重复序列。

### 8.Experimental infection of piglets

**仔猪实验性感染**
- [ ] 动物实验得到了国家有关部门门的批准(证书SYXK(Hu)2007-0025)。四头50到60天大的猪在基因上与原始病猪相似，它们是从商业牛群中购买的。
- [ ] 进行PCR检测，以证实PSV的缺失。
- [ ] 这些母猪被安置在一个动物设施中，并被允许接触食物和水。观察1周后(确保无症状)，随机分为两个治疗组之一(每组2头)。.
- [ ] A组每头猪口服PSVCSH株4ml,滴度约为107.5 TCID 50/ml。假手术对照组B组给予4ml MEM。
- [ ] 接种后，每天监测动物是否有明显的临床症状。每头猪在接种后第0、1、3、5、7和9天采集粪便样本。
- [ ] 在第9天时，所有动物均被出血杀死，并进行病理检查.将肠、肺、脑固定在10%中性福尔马林溶液中，常规处理，石蜡包埋。
- [ ] 取5 Im厚的石蜡切片,用苏木精和伊红(HE)染色进行组织病理学检查。
- [ ] 同时，将滤过的粪便悬液或组织匀浆接种到PK 15培养物上，经CPE和RT-PCR鉴定，从粪便和肺中分离出病毒。

## Results

### 1.Pathogen examination

**病原检查**
- [ ] 对17头病猪进行了病毒、粪便和器官标本的PCR/RT-PCR检测，以确定猪体内是否存在PoRV-A、PoRV-C、 PEVB、猪杯状病毒、PAstV、PPV、PCV 2、PRRSV. PRV.CSFV. TGEV. PEDV、 PTV和JEV。 除-份PCV 2阳性样本外,所有检测结果均为阴性。
- [ ] 然后对所有样本进行PSV检测，结果显示所有样本均为阳性。
- [ ] 所有样本中常见致病性肠道细菌，如致病性大肠杆菌、沙门氏菌、空肠弯曲菌和C型产气荚膜梭菌均为阴性。
- [ ] 对不能通过表型和生化特征鉴定的细菌进行进一-步PCR检测，所有标本均为支原体、钩端螺旋体、链球菌、李斯特菌和布鲁氏菌阴性。

### 2.Virus isolation

**病毒分离**
- [ ] 在pK- 15细胞上传代5次，观察到明显的细胞病变效应.接种2~3天后，首次观察到细胞的膨大、包围和脱落，第4天细胞与培养瓶表面分离。
- [ ] 进-步传代后,病毒滴度为107.5 TCID50/ml.

### 3.Morphological characteristics

**形态特征**
- [ ] 用电镜观察了该病毒的形态特征。用电子显微镜观察，发现有直径约28~30nm的小球形病毒颗粒。在某些病毒子中，二十面体的结构是明显的(图1)。

### 4.RT-PCR detection and sequence analysis

**RT-PCR检测及序列分析**
- [ ] 为确定细胞培养上清液(第8代)和自然感染仔猪原始病料中电镜下观察到的病毒是否为PSV，采用PSV特异性引物进行RT-PCR。
- [ ] 琼脂糖凝胶电泳后，观察到约180 bp的扩增产物.然后对该带进行纯化和测序。结果表明，获得的序列与PSV V13株(GenBank ID: AF 40681 3)的50-UTR同源性为90%。
- [ ] 为进一步鉴定分离的病毒， 采用10套针对PSV V13株基因组设计的引|物和50~30RACE法测定了CSH株的全基因组序列。
- [ ] 该病毒株(GenBank ID: HQ 875059)全长7502 nt,含有6, 996-nt开放阅读框(ORF),编码2，331-AA多蛋白前体。这个ORF序列前面有至少441 nt长的50UTR，然后是至少65 nt长的30 UTR.
- [ ] 以CSH株P1基因区和9个Picornavirus属28种典型病毒为基础，构建了一株系统发育树，对CSH株的系统发育进行了研究。
- [ ] 结果表明，CSH菌株与PSV菌株po 5116和Sek 1562/98有密切的聚类关系，并与其它PSV毒株形成一个明显的分支，与Picor导航科的其他成员分离(图2)。

### 5.Experimental infection, necropsy, and histopathological examination

**实验性感染、尸检和组织病理学检查**
- [ ] 2日龄时出现腹泻和呼吸著迫综合征(A组)，7天出现以共济失调和腿部麻痹为表现的脊髓灰质炎。这些症状与临床观察的相似。
- [ ] 从接种猪的粪便和肺标本中分离到PSV,并对其进行鉴定。B组对照组无症状。
- [ ] A组猪的尸检显示病变主要集中在肠、肺和脑。小肠粘膜广泛充血(图3-2)。肺下叶部为实变，左肺叶可见大量出血点(图3-2)。
- [ ] 此外，硬脑膜出现点状出血和充血(图3-2)。B组对照猪无明显病变(图3-1).组织病理学检查。
- [ ] 小肠各部位均有明显的绒毛脱落，小肠固有层有出血性浸润(图4-1)。肺间质和肺泡内含有大量红细胞。
- [ ] 此外，肺泡扩张和肺泡壁变薄，部分肺泡破裂成为大囊肿(图4-2)。脑出现一些特征性的神经元空泡化和血管周围的袖带(图4-3).
- [ ] B组对照组无症状(图4)。

## Discussion

- [ ] 对_上海某猪场猪急性腹泻、呼吸窘迫和脊髓灰质炎的病因进行了诊断性调查。
- [ ] 我们对患病猪的粪便和器官进行了病毒和细菌测试，这些病毒和细菌在猪身上引起了类似的症状。根据检测结果，标本中唯一一致检测到的病毒是PSV.
- [ ] 所有标本均为PSV阳性，1例PCV 2阳性，但未检出其它可能的病毒(包括PRRSV. PRV.CSFV、PoRV-A、PoRV-C、 PEVB、 猪calicivirus、PAstV、 PPV、 TGEV. PEDV和JEV)或细菌(包括致病性大肠杆菌、沙门氏菌、空肠弯曲杆菌、产气杆菌C型、支原体、Leptosa、链球菌、李斯特菌和布鲁氏菌)。
- [ ] 用pK-15细胞从阳性标本中分离PSV.经过五次实验，我们观察到了一个典型的CPE,这与前面对PEV-A的描述是一致的。
- [ ] 然后，用RT-PCR方法鉴定该分离物为PSV,并对自然感染仔猪的原始标本进行鉴定。
- [ ] 此外，电子显微镜观察了该病毒的形态，但还没有确切的描述PSV粒了子结构的报道。电子显微镜显示PSV是直径约28-30 nm的小球形颗粒。
- [ ] 在一-些病毒子中，二十面体的结构是明显的(图1)。这些形态特征与其他昆虫科成员的形状和大小是-致的。此外，分离出的病毒颗粒多为固体颗粒，说明该分离物可能具有较好的传染性。
- [ ] 为了进一步鉴定病毒分离物，测定了CSH株的全基因组序列。.该病毒的全长基因组为7502nt,含有-个编码2, 331-aa多蛋白前体的单- -ORF.
- [ ] 当使用BL AST搜索程序时，该分离物与PSV-V1 3株的核苷酸序列同源性为85%，病毒基因组的基本组成为: A 30.38%，C18.98%，G 22.42%.U 28.22%。
- [ ] 该PSV菌株的多蛋白前体编码一个预测的84-aa先导蛋白，并分别编码VP1、VP2、VP3和VP 4结构蛋白53、238、 234和285 aa.
- [ ] 非结构蛋白的预测长度2A、2B和2C分别为234 AA、105 AA和332 AA,而3A、3B、3C和3D的预测长度分别为100 AA、22AA、182 AA和462 AA.
- [ ] 这些特征与前面描述的PSV的遗传结构是一致的。有趣的是，与PSV-V13菌株相比，该序列在30端缩短了17 nt,编码多聚蛋白前体的ORF长了31 nt.
- [ ] 这种差异的功能意义需要进一步研究。
- [ ] 进一步根据CSH株P1基因序列进行系统发育分析，证实该分离物与欧洲PSV分离株po511 6和Sek 1562/98密切相关，从而与其他PSV毒株构成一个亚组，与Picor导航科的其他成员分离(图2)。
- [ ] 这些结果表明，分离出的病是种与欧洲PSV毒株关系最密切的猪脑病毒。
- [ ] 最后，进行动物接种实验，观察猪感染PSV后的临床表现和病理变化。
- [ ] 第2天出现腹泻和呼吸窘迫，第7天出现脊髓灰质炎综合征(共济失调和腿部麻痹)。这些症状与临床受试者所观察到的症状以及由ptv弓|起的临床症状相似。
- [ ] 尸检结果显示病灶主要见于肠、肺和脑(图3)。
- [ ] 有趣的是，尸检中发现的皮损与感染PTV的猪相似,这表明PTV也可能在混合感染中出现PSV.组织病理学检查。
- [ ] 小肠和肺的组织病理学变化(图4)与先前报道的感染ptv和其他肠道病毒的猪相似(图4)。
- [ ] 在接种猪的脑组织中，发现PTV和其它肠道病毒感染的主要神经病理特征为神经元空泡化和血管周围袖带(图4)。
