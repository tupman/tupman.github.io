## Abstract

- [ ] 猪智齿病毒(PSVS)广泛分布于猪群中，但对猪的进化历史及其分化的机制了解甚少。
- [ ] 因此，本研究对湖南26个猪场的241份粪便样本和91份肠道内容物进行了检测，以确定:是否存在PSVS。PSV总阳性率为46.39%，其中苗重和育肥猪的感染率较高。
- [ ] 共分离到29株PSV(PSV-Huns)菌株，经系统发育和配对距离分析，显示出较高的遗传多样性。衣壳蛋白-与3CD区的亲缘关系树间的不- -致表明PSV-Huns内存在频繁的重组，并在P1区3端附近发现了一个可能的重组热点。
- [ ] 结果表明，重组在PSV遗传多样性和进化过程中起着重要作用。

## 正文

- [ ] Sapelovirus A,原为猪Sapelovirus(PSV),隶属于Picor导航科Sapelovirus属(Sapelovirus)。
- [ ] PSV基因组由一个长度约为7 .5kb的线性、阳性、单链RNA分子组成，并包含一个大的ORF.它被翻译成-个单- -的多蛋白。
- [ ] 多蛋白在翻译后被病毒编码的蛋白酶分阶段切割，产生12个成熟蛋白，包括一个领导蛋白(L)、四个结构蛋白(VP1-4)和七个非结构蛋白(2A-C, 3A-D)。
- [ ] 现场的PSV感染可能导致无症状或症状性感染。在前一种情况下，PSV与其他肠道病原体的混合感染是常见的。
- [ ] 最近发生的PSV感染病例与不典型的神经症状有关，包括脊髓损伤、食欲不振、活动障碍、共济失调、不协调、精神迟钝、瘫痪、瘫痪以及对环境刺激的反应减少。
- [ ] 然而，实验性PSV感染与广泛的症状有关，包括腹泻、呼吸砉迫和脊髓灰质炎，表现为共济失调和腿部麻痹。许多国家都报告了PSV感染。
- [ ] 中国第一株有记载的PSV株(CSH)是在2009年发现的，近年来又有报道.
- [ ] 虽然VP1基因[16-18]的系统发育分析表明了PSV株间的遗传多样性，但导致PSV进化的机制尚不清楚。
- [ ] 因此，本研究的目的是通过对湖南分离的PSV流行和基因组的分析，阐明PSV进化的机制。
- [ ] 获得了接近完整的基因组序列，并与GenBank中其他已知的PSV序列进行了比较。
- [ ] 通过系统发育和两两距离分析来确定它们之间的遗传关系，并进行重组分析以阐明导致遗传多样性模式的潜在因素，进而决定PSV进化的本质。
- [ ] 为了解湖南省PSV流行情况及分子遗传关系，于2014年2月至2017年4月，在湖南省26个猪场采集了241份粪便样本和91 份肠道内容物。
- [ ] 在样本中，101份来自年龄介乎5至70天及有腹泻史的猪只，其余231份来自无临床征状，年龄介乎5至140天的猪。
- [ ] 将0.5g等量粪便在1ml冰冷Roswell Park Memorial Institute(RPMI)1640培养基(GE Healthcare Life Sciences，匹兹堡，美国宾夕法尼亚州匹兹堡)中匀浆，其中含有1%青霉素和链霉素(P和S)，在球磨机(Retsch，Haan，德国)中匀浆，并在12000g下离心20min。
- [ ] 1 2 000 g离心20 min,直接澄清肠内容物
- [ ] 将部分上清液(0.2ml)转入2 ml Eppendorf管，用于提取病毒RNA(北京，中国),其余上清液保存在80°C，直至使用。
- [ ] 根据厂家的指示，利用Revertaid FirstStrand cDNA合成试剂盒(Thermo FisherScience, Waltham, 美国)用随机引物合成病毒cDNA。
- [ ] 从GenBank下载已知的PSV基因组序列，用Lasergene软件包(DNAStar, Madison,WI, USA)进行分析。
- [ ] 设计了一对引物(PSV-fp: 5-CGTGCTCCTTTGTGATTC-3; PSV-RP: 5-GAGAGTAGTATATATATATATATATCC-3)，该引物位于保守的5%UTR中，目的是扩增PSV的270 bp片段。
- [ ] PCR在以下条件下进行: 94°C初始变性5min, 94C35次30s，53C 30s, 72C 15S,72°C最后延伸7 min.用0.8%琼脂糖凝胶电泳分析PCR产物，并在紫外线照射下观察溴化乙锭染色后的结果。
- [ ] 用SPSS软件19.0(ibm，美国)进行卡方检验,调查PSV流行率的差异。
- [ ] 结果在P<0.05时被认为是显著的。
- [ ] 332份标本中有154份检出PSV,总阳性率为46.39%。PSV检出率分别为13.1696(15/114)、60.15%(80/133)和69.41%(59/85)。育苗育肥猪PSV感染率较高(P<0.01)，显著高于乳猪(P<0.01)。
- [ ] 然而，PSV在不同品种的猪中普遍存在，说明PSV很可能长期与宿主共存。
- [ ] 此外，无腹泻症状的乳猪感染率(21.57%)显著高于腹泻仔猪(6.35%)，但无腹泻症状的仔猪感染率(60%)和腹泻仔猪的感染率(60.539%)无显着性差异(表1)。.
- [ ] 这些结果与以前的研究结果-致, 表明PSV在腹泻猪中广泛流行。
- [ ] 然而，有趣的是，非腹泻猪的感染率(54.98%)明显高于腹泻猪(26.73%)，说明在某些情况下，PSV可能不是致病性的，也可能对宿主没有影响。
- [ ] 此外，我们亦详细调查了在两个商业农场及无腹泻史的猪只中，年龄介乎2至20周龄的猪群感染PSV的情况。特别是在8至16周的猪体内，PSV感染率更高(表S1,见网上补充材料)。
- [ ] 将136份粪样和1 8份经鉴定为PSV阳性的肠道内容物的上清液通过消毒的0.22-m滤池(PALL,华盛顿港，NY,美国)，在含有1%P和S的RPMI-1640培养基中稀释1: 10, 并与猪肾单层细胞(PK-15)在24良好塑料培养板(Corning, Corning, NY, USA)中孵育1h,在5%CO2中培养37°C。
- [ ] 添加新鲜RPMI-1640培养基/2%新生小牛血清(Invitrogen, Carlsbad, CA,美国)/1%P和S，并继续孵育4天。
- [ ] 将培养物冷冻解冻三次，然后在pk-15细胞中传代。总之，其中29个样本表现出细胞病变效应,其特征是细胞收缩、四舍五入和细胞在三次传代后脱落。这些培养上清液被收集并保存在80°C直到需要。
- [ ] 从上清液中提取病毒RNA，并根据制造商的指示使用Revertaid First Strand cDNA合成试剂盒(Thermo Fisher Science)进行逆转录。
- [ ] 应用PFU DNA聚台酶(TSINGKE,北京,PR)对cDNA进行PCR扩增。设计了五个引物集(表S2),根据已公布的PSV参考株基因组序列，扩增每个已鉴定的PSV株的多蛋白基因重叠区(表S3)。
- [ ] PCR产物经琼脂糖凝胶电泳分析后测序(TSINGKE)。对29株PSV(暂命名为PSV-HuN5-33)的近全基因组进行测序，并提交GenBank(登录号)。(f 440633-MF440661)
- [ ] 通过对29株pSV-Hun株的编码序列进行比对和蛋白酶切割位点的预测,发现编码2323-2334-氨基酸(Aa)的多蛋白基因组序列包含6972~ 7005个核苷酸，大部分pSV-Hun株含有6996个核苷酸，编码2332-aa序列。
- [ ] 编码序列的差异与VP1 C端区发生的AAS的插入或缺失显著相关，与所有其他具有最常见基因组组织的PSV序列比较有显着性差异(图一)。
- [ ] VP1-vp3的C端位于病毒表面，与每-种p1vp3的形态和抗原性有关。
- [ ] 特别是, vp1的C端在许多微微病毒中诱导中和抗体,从而表明暴露的抗原SIT的存在.
- [ ] 菌株HuN 21和HuN 32含有4个AAS(STAE)插入到AA 283和AA 284之间，另一个脯氨酸插入到VP1的AA 286和AA 287之间，这与我们以前的REPO-致。
- [ ] 株HuN6还含有三个额外的核苷酸，对应于插入在AA286和AA287之间的一个额外的亮氨酸。
- [ ] 菌株HuN 9、HuN 22和HuN 31在AA 282和AA 290之间存在7-AA缺失，HuN 12在AA 282和AA 291之间表现出8-AA缺失(S1a)。
- [ ] 此外，菌株hun 21和hun 32在vp3的aa 57和aa 60之间存在一个2-aa缺失，这在所有其他已测序的PSV菌株中是独一无二的。
- [ ] 在系统发育分析方面，在DNAStar软件中使用集群W程序对一组序列进行了比对。
- [ ] 系统发育分析由野兽1.8.1采用贝叶斯推理(BI)方法进行，mega Version6.06(www.megas oftware.net/)通过极大似然(ML)方法进行分析。
- [ ] 使用jModeITest版本2.1.10确定了每个数据集核苷酸的最佳拟合模型。
- [ ] 特别是在核苷酸序列分析方面，BI方法采用具有不变位点比例和γ-分布速率异质性的广义时间可逆(GTR)替换模型(GTR+G+I),用GTR+I构造具有1000个自举复制的ML树，以检验系统发育。
- [ ] 在BI分析中，使用了-个不相关对数正态分布和链长为1 108代的松弛分子钟，每1000代采样- -次，以扩展的贝叶斯天际线图作为树的先验，并将刻录设定为采样状态的10%。
- [ ] 用程序Tracer(http://beast.bio.ed.ac.uk/tracer/))评估这些树，并通过图3(http://tree.bio.ed.ac.uk/software/figtree/).)观察这些树
- [ ] 为了表征PSV-Huns的遗传差异，利用mega6.06软件，用未修正的p-距离法计算了pSV-Huns的遗传距离。
- [ ] 皮卡那病毒的衣亮含有四种结构蛋白(Vp1-4),是免疫显性和高度可变的。
- [ ] 基于结构蛋白序列的系统发育分析被广泛应用于微微病毒属的分类研究。衣壳蛋白基因序列的系统发育分析表明，利用BI方法(图1A)，根据进化树的拓扑结构,将所有的PSVS分离成至少两个聚类。
- [ ] 大多数PSV-Hun菌株与先前报道的亚洲(中国和韩国)PSV-起属于第1组，而HuN 23和先前报道的PSV-Hun株(HuN 2- 4)和株HuN9、HuN 12、HuN 22和HuN 31与美国的一株密切相关的菌株(ISU-SHEIC)-起被分离到第2组。
- [ ] 此外，在PSV-Huns之间, Capsidprotein基因序列的配对距离在0.001~0.231之间.这些数据揭示了湖南PSVS的巨大遗传多样性。
- [ ] 然而，对PSVS所有有效3CD基因序列的系统发育分析表明，与衣壳蛋白基因序列相比，具有明显的系统发育拓扑结构(图1B)。
- [ ] 大多数菌株在这些不同基因组区域的系统树上都发生了洗牌，表明衣壳蛋白和3CD区是独立进化的。在其他肠道病毒中，不同基因组区的独立进化已经被报道，表明重组在不一致的树状结构中起着重要的作用。
- [ ] 因此，我们推测PSV-Huns中也存在潜在的重组事件.
- [ ] 为了检测已鉴定的29株PSV-Hun株的全多蛋白基因序列中可能发生的重组事件，利用重组检测程序(RDP)4.84[43]中的7种方法，包括RDP、GeneConv、 BootScan.Maxchi、Chimaera、 SiScan和3 Seq。
- [ ] RDP还定义了断点和相似点。
- [ ] 利用SimplotVersion3.5.1软件对可能的重组菌株进行了预测，确定了支持相关树拓扑的系统发生信息位点。重组和突变对病毒的进化起着重要的作用，重组事件在微微病毒中已经被广泛观察到。
- [ ] 在我们之前的工作中，psv报道了一个重组事件。
- [ ] 在本研究中，我们利用RDP和Simplot程序对29个PSV-Hun毒株和其他已知的PSV毒株的多蛋白基因进行了重组分析。标准相似图分析采用Simplot程序和HuN 6.HuN 16、HuN 23、HuN 24和HuN 26序列作为单独查询。
- [ ] HuN 6在P1区与HuN 17具有高度的同源性，P2和P3区域与HuN 7的核苷酸序列高度相似(图2A)。HuN 23在P2和P3区域与HuN 22具有显著的序列同源性，而在HuN 23 P1区存在高度相似的潜在亲本菌株(图2C)。
- [ ] HuN 24在P2和P3区域与HuN 22有很高的相似性，而在P1区与HuN 24具有显著同源性的潜在亲本菌株(图2e)无法鉴定。
- [ ] HuN16分别在P1区和2C区的2A-2B-5±一半与HuN7和HuN12高度相似，而在2C和P3区的3±一半与HuN16没有发现高度相似的潜在亲本菌株，这表明其进化历史上发生了不止一次重组事件(图2G)。
- [ ] HuN 26在P1和P3区域与HuN 1 9有很高的同源性，在P2中间区域与HuN 12具有显著的同源性，而可能与带有HuN 16的P2其他区域具有高度相似性的亲本菌株无法被识别，这也表明在其进化历史上有多个重组事件(图2i)。
- [ ] 然后，我们进行了引导扫描分析，以确定重组子中的潜在断点(图2B，d，f，h和j)。
- [ ] 在5个重组子中，在p1区3端附近发现了-一个可能的重组热点，与肠病毒基因组中编码vp1C端区域的潜在重组热点一致。目前，只有少数几个PSV被完全测序，这为检测重组事件提供了极小的机会。
- [ ] 与PSV株相关的重组规律可望在获得和测序循环型PSV株的额外完整基因组后得到完全阐明。
- [ ] 总之，PSV在湖南猪中广泛流行，在8~16周龄无症状猪中的感染率特别高。
- [ ] 从中国中南部分离到的29株PSV-Hun菌株，通过对衣壳蛋白基因序列的系统发育和配对距离分析，表现出较高的遗传多样性。
- [ ] 衣壳蛋白与3CD区的亲缘关系树间的不- -致表明PSVHuNs内有频繁的重组，并在P1区3端附近发现了一个可能的重组热点。
- [ ] 然而，在我们的研究中，用于调查PSV流行的样本数量是有限的，在我们未来的研究中将处理更多的样本。此外，这一潜在病原体的致病性和临床影响还没有得到详细的研究。
- [ ] 因此，计划进一步研究这29株PSV的致病性，以防止这种潜在病原体的爆发，井避免对猪肉行业造成经济损失。
