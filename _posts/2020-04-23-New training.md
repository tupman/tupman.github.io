## 焦法梅《链家管理制度》

- [ ] **2.升降级标准和积分计算**
- [ ] **转正的考核标准：**
- [ ] 三个月内完成20000元业绩，200积分，即可转正，未转正则延期3个月为400分
- [ ] 每月21日前计入考核
- [ ] 新人基础分为0，从A0/2、A1起基础分为50，之后逐级增加50
- [ ] 期末积分=当期业绩*1%-基础分+期初积分+当期奖励积分-当期处罚积分
- [ ] **3.底薪加提佣**
- [ ] A0A1提成30%A2 A3提成35%，之后逐级增加5%
- [ ] 最高M10为75%，A10为70%
- [ ] ②业绩分配的角色认定标准：
- [ ] 录入人，维护人，钥匙人，实勘人，委托书人
- [ ] 经纪人获得房源，1小时内录入系统
- [ ] 维护人，对某房源按照公司标准进行跟访维护
- [ ] 实勘人对房源进行实地勘察
- [ ] **注意**：不能上传双水印的实勘照片，否则为红线
- [ ] 钥匙人，收取房源钥匙，并与业主签订钥匙托管协议，在系统中登记钥匙
- [ ] 房源委托书人，需上传房屋出租委托登记表，房屋状况说明书服务事项告知书
- [ ] ③业绩分配原则：17年8月1日起，83.96%
- [ ] **买卖**，录入人10%、维护人20%、实勘2%、拍照3%、钥匙5%、推荐5%
- [ ] **租赁**，录入人10%、维护人5%、委托书人10%、实勘人5%、钥匙人10%、推荐人5%
- [ ] ④管理制度：
- [ ] 日常考勤，红线、黄线、黑线

## 《房地产基础知识》

- [ ] **1.基础**
- [ ] ①房地产三要素：
- [ ] 实物-物理属性
- [ ] 权益-权利属性
- [ ] 区位-空间属性
- [ ] ②房地产市场三要素：
- [ ] 市场主体，买卖双方
- [ ] 市场客体，商品本身
- [ ] 交易条件，成熟交易条件
- [ ] ③房地产市场特性：
- [ ] 交易对象不能移动，
- [ ] 交易对象各不相同，
- [ ] 交易金额较大，
- [ ] 交易过程较长，
- [ ] 交易频次较低
- [ ] ④房地产市场分类：
- [ ] 一级市场，土地使用权出让市场，国家参与的政府性地产转换
- [ ] 二级市场，房地产商个人行为或两个房地产商间的行为
- [ ] 三级市场，存量房市场
- [ ] ⑤土地使用权：个人单位依法获取占有使用收益有限处分的权益
- [ ] ⑥依法获取的方式：
- [ ] 出让包括，招标出让、拍卖出让、挂牌出让、协议出让
- [ ] 划拨包括，无偿交付/缴纳补偿/、比如政府军队用地
- [ ] ⑦根据出让年限分类：
- [ ] 70年纯居居住用地，
- [ ] 50年公用用地，科教文体卫生体育用地，综合或其他用地
- [ ] 40年纯营业设施的就是40年，比如商业旅游娱乐
- [ ] ⑧根据建筑用途分类：
- [ ] 居住建筑包括住宅公寓和集体宿舍
- [ ] 公共建筑，农业建筑，工业建筑
- [ ] ⑨建筑构造构成：
- [ ] 竖向建筑构件，墙体基础和柱
- [ ] 水平建筑构件，地面、楼板、梁、屋顶
- [ ] 解决上下层，楼梯
- [ ] ⑩墙体：
- [ ] 承重墙和非承重墙
- [ ] ①基础：
- [ ] 建筑物地面以下的承重结构，如基坑承台框架柱地梁
- [ ] ②建筑构造名词解释：
- [ ] 梯户比，七层以上超过16米必须有电梯，12层以上不少于两部
- [ ] 建筑层数，分为地上和地下层数，层高少于2.2米不算
- [ ] 自然层数，超过2.2米的半地下室也算
- [ ] 设备层
- [ ] 地下室，地平面超房间总高的1/2
- [ ] 半地下室，地平面小于房间总高的1/3
- [ ] 阳台，露台，飘窗
- [ ] **2.房地产产品分类和房地产开发**
- [ ] ①房屋按照销售阶段分类：
- [ ] 期房，房新建房从开发商取得商品房预售许可证智取得初始产权证期间都称为期房，期房销售时需要五证齐全
- [ ] 现房，通过竣工验收，交付使用，并获得房屋产权证的房屋现房销售时需要五证两书齐全
- [ ] 尾房，楼盘已销售80%房源后剩余的
- [ ] ②按照高度分类：
- [ ] 超高层是40层以上，超过100米
- [ ] 高级住宅10层以上，小高层8~11层
- [ ] 中高级住宅7到9层
- [ ] 多层住宅4到6层
- [ ] 低层住宅1到3层
- [ ] ③房屋按照建筑类型分类：
- [ ] 板楼，塔楼
- [ ] 结合，独栋，叠拼
- [ ] ④房屋按照室内平面结构分类：
- [ ] 平层建筑所有功能区都在同一平面，常见的住宅多以平层为主
- [ ] 越层建筑跨越两楼梯以上，各层之间相对独立
- [ ] 错层建筑功能区不在同一平面，各功能区有高度差
- [ ] 复式建筑层高较高，局部掏出夹层楼梯相连不是完整的两层
- [ ] ⑤按照房屋产权性质分类：
- [ ] 经济适用房，房改房（公房），央产房，限价商品房，廉租房，定向安置房，军产房
- [ ] 除普通商品房外，以上均不能直接上市交易，需要办理相关手续才可
- [ ] ⑥按照房屋居住舒适度分类：
- [ ] 普通住宅，住宅容积率1.0以上，建筑面积140平以下，单价不超过同环最高线，以上三点需要同时满足为普通住宅
- [ ] 非普通住宅，公式：5环以内\*0.8=5-6环，  5环*0.75=6环以外
- [ ] 5环内小于39600，每套小于4685000
- [ ] 5到6环，小于31680，小于374.44万
- [ ] 六环以外，小于23760，小于280.8万每套
- [ ] **⑦房地产开发流程及相关证书：**
五证两书，包括国有土地使用许可证，建筑用地规划许可证，建筑工程规划许可证，建筑工程施工许可证，商品房销售许可证，住宅质量保证书，住宅使用说明
- [ ] **3.房地产的法律法规**
- [ ] 定金具有惩罚性，是对违约行为的惩罚
- [ ] 收定金一方不履行债务时，双倍返还定金
- [ ] 定金数额不得超过主要额的20%，
- [ ] 法律效力，定金＞订金
- [ ] **4.房地产专有名词解释：**
- [ ] 生地，毛地，熟地
- [ ] 容积率，地上总建筑面积 / 建筑用地面积
- [ ] 绿地率，各类绿地面积的总和除该地区总面积
- [ ] 绿化率，绿化覆盖面积除该地区总面积，或者绿化种植垂直投影面积除该地占地面积
- [ ] 绿化率大于等于绿地率
- [ ] 0.5以内独栋或双拼别墅
- [ ] 0.5到1.0，一般的双拼联排别墅
- [ ] **5.房屋测绘名词：**
- [ ] 层高，净高，进深，开间（阳光方向）
- [ ] 住宅层高不应小于2.7米，净高等于层高减楼板厚
- [ ] 得房率等于套内使用面积/建筑面积

## 杜佳旺《快速掌握基础业务流程》

- [ ] **1.业务简介**
- [ ] ①租赁业务的特点：
- [ ] **短**，客户找房周期短，考虑时间短，流程周期短；
- [ ] **频**，消费频次高，带看；
- [ ] **快**，租赁更注重速度，短时间内考验你的精准度
- [ ] **②业务流程：**
- [ ] **了解业务特点**，利用店面接待、社区驻守、内部员工介绍、link消息、400接听等渠道开发，
- [ ] **房客源开发**，留存业主身份证明复印件、留存房屋权属证明的复印件、与业主签订
- [ ] **需求理解**，了解客户购房动机、业主出租要求
- [ ] **匹配**，了解需求后为房找客、为客找房
- [ ] **带看**，匹配成功后、带客户实地看房
- [ ] **斡旋**，客户对房子有意向、三方斡旋
- [ ] **签约**，客户业主均满意、签署合同
- [ ] **物业交割**，陪同交水电、燃煤气等
- [ ] **成交后维护**，感情维护
- [ ] **2.关键业务动作及常见问题**
- [ ] **业主在意的内容**，出租价格高越好、租客质量爱惜房屋、省心出租
- [ ] **经纪人业务动作**，实地评估了解房源，抓住优势，迅速传递，多带看了解客户情况，为业主把关、收钥匙推荐自如收房
- [ ] **客户在意的内容**，房租便宜，房源真实，可看多套付款省心
- [ ] **经纪人的业务动作**，了解房屋情况，协助沟通业主，避免涨价，提供真房源，提供更多房源选择，为客户匹配房源，了解业主收款条件
- [ ] ①匹配思路局限客户流失**解决**：
- [ ] 带看后一定要跟客户当天回访、澄清需求、带看后要约定好二次服务
- [ ] 行程量越多，成交的可能性就越高，活动量有助于产生有效行程量
- [ ] ②有效行程**五大指标**：
- [ ] 客源端包括新增客源量、客源带看量，
- [ ] 房源端包括新增房源量、达标实勘量、新增钥匙量，
- [ ] 新增客源量，一时间匹配客源录入
- [ ] 新增房源量，房源录入分得10%业绩
- [ ] 客源带看量，带看是接近开单的环节、是开单的希望

## 曾小钱《商圈跑盘与空看》

- [ ] **1.深入了解商圈及楼盘，跑盘**
- [ ] ③跑盘对经纪人的帮助：
- [ ] 一熟悉商圈，你能清楚商圈的道路情况方便带看、能够记住周边的配套
- [ ] 二熟悉楼盘，熟悉楼盘配套、提升专业
- [ ] 三熟悉同事，熟悉商圈内的链家店面、认识同事
- [ ] **④跑盘前的准备工作：**
- [ ] **物料**准备，背包铅笔、橡皮笔记本、水杯
- [ ] **心态**准备，辛苦是有的（画图）
- [ ] **信息**准备，询问商圈经理或师傅、跑盘的范围、时间和交付物（商圈跑盘图、）
- [ ] ⑤店面所在商圈信息的采集：
- [ ] 记住商圈内所有道路名称、红绿灯口堵车情况
- [ ] 各楼盘名称、位置
- [ ] 链家店面位置、店名
- [ ] 记录每条路的周边配套
- [ ] ⑥周边大型配套要采集的信息：
- [ ] **包括商业、银行、医疗机构、学校、交通、公园景点**
- [ ] 商场要记录下面的位置和名称，酒店超市商场大型标志性建筑娱乐场所（地标、KTV、写字楼）
- [ ] 银行要采集两个信息，各大银行营业厅名称位置、ATM机位置
- [ ] 医疗机构采集的信息，位置等
- [ ] 学校要采集的信息，学校位置、名称、入口（招生条件、规模、年级、知名度、收费标准、教学特色、放学时间）
- [ ] 交通采集的信息，公交站点地铁站位置站名、附近有哪几路公交车、地铁站几号线（岁数大的人经常坐公交）
- [ ] 公园景点要采集的信息，名称位置、收费情况、开园闭园时间、有无特殊活动（庙会）
- [ ] ⑦商圈包含楼盘信息的采集：（**跑盘结束后及时返回店面画商圈图、各楼盘平面图、主动让师傅帮你核查**）

## 陈鑫《业务流程之实勘和钥匙》

- [ ] **1.成为房屋的代言人，实勘**
- [ ] ①什么是实勘？（了解业主想法）
- [ ] 是服务流程的重要一环，经纪人要实地勘察房屋、获取房屋信息
- [ ] 确定房源真实性、避免交易风险
- [ ] 增加与业主交流机会、挖掘业主潜在要求、加速房屋出租、签约

- [ ] 了解详细信息、了解业主心里状态

- [ ] 展示自己的专业、赢得业主好感、获得业主信任、成为专属经纪人
- [ ] ③实勘前的准备：
- [ ] **房屋准备**，
- [ ] 与业主确定实勘时间、把事情精简、和摄影师商量好、
- [ ] 业主准备房屋相关证件、
- [ ] 请业主提前整理和打扫房屋、

- [ ] 事前联系好摄影师、
- [ ] 必备工具、包括鞋套、名片、钥匙托管协议、出租委托协议
- [ ] 加分工具、历史出租记录（更专业）、
- [ ] **2.实勘中的信息获取**
- [ ] ①信息有三类：
- [ ] 房屋的基本信息-物理属性
- [ ] 证件出租条件-法律属性
- [ ] 出租原因急迫度-心理属性
- [ ] **物理属性：**
- [ ] 记录准确的房屋地址，建筑结构总层数、房屋性质设计用途、建筑面积所在层数
- [ ] 查看房产证上的户型图，
- [ ] 查验房屋的实际户型，记录有无改造情况、询问各个功能间的朝向
- [ ] 回店后上传优质户型图，
- [ ] 了解房屋面积，看房产证上的精确面积、建筑面积、套内建筑面积、记录或测量各个功能间的面积
- [ ] 了解房屋的装修程度，了解装修和家具家电
- [ ] 了解房屋使用细节，电闸位置、水电燃气缴纳方式、厨房卫生间整洁程度、重点关注低楼层的采光
- [ ] **法律属性：**
- [ ] 获得房屋权属属性为后期交易做准备
- [ ] 房产证原件，
- [ ] 房屋的产权性质，只有商品房、已购公房、回迁房可以出租
- [ ] 是否有共有权人，都要签字
- [ ] 是否为代理人签约，
- [ ] 近期内是否打算出售？（为客户提前解释用）
- [ ] **心理属性：**
- [ ] 深入了解业主内心想法
- [ ] 出租原因，可入住的时间、出租的年限（询问业主是否长租、以及帮助搬家）
- [ ] 资金需求，
- [ ] ②实勘照片的标准：
- [ ] 画面内容，无人物、画质好、空间感（五四三墙面定律）、避私密（盖上马桶盖）、不失真、无水印、不重复
- [ ] 片设置，
- [ ] 户型图、绘制正确功能间、位置正确
- [ ] 图片数量与设置、
- [ ] 拍摄高质量实勘照片：
- [ ] 光线要好，正确对焦
- [ ] 角度合适，面积大的房间利用对角线五面取景、面积小的房间半蹲、保持四面取景
- [ ] **3.实勘后的出租建议**
- [ ] ①报价建议：
- [ ] 分析房子的优劣势，位置户型、装修等
- [ ] 分析市场行情，同类户型出租情况、库存情况、市场行情
- [ ] 分析业主情况，是否急租、付款方式
- [ ] ②看房时间建议：
- [ ] 商议固定时间，方便客户看房
- [ ] 最佳带看时间，朝南房屋在中午11点到下午两点、朝西向房屋在15点到17点
- [ ] ③客户选择建议：
- [ ] 哪些方面分析客户？租房原因、急迫度、居住人群、付款方式
- [ ] 根据业主需求给出建议，对爱护房屋有要求、对付款方式有要求
- [ ] ④美化建议：
- [ ] 物品摆放设计，添加DIY小样式等
- [ ] **4.打开成交之门-钥匙**
- [ ] ①房屋适合推荐钥匙：
- [ ] 不方便过来的业主，平时忙于工作，经常出差的业主，
- [ ] 空置房源的业主，
- [ ] 委托一段时间后仍未出租的业主，
- [ ] 急租需要通过钥匙增加带看量，
- [ ] ②争取为业主提供钥匙服务：
- [ ] 了解业主情况，
- [ ] 工作性质、居住离房源较远、是否急租、
- [ ] 看房需要经常配合开门、需要时间和精力、
- [ ] ③提供钥匙服务，**服务内容**包括：
- [ ] 随时带看加速交易，经纪人介绍传播房源可带看时间长、不错过客户
- [ ] 物品保管和登记，签订协议
- [ ] 房屋维护和保持，带鞋套看您的房子、定期通风、打扫卫生
- [ ] **5.作为经纪人，能为业主做出承诺：**
- [ ] 立刻进行出租动作，专属维护服务，热情的看房服务
- [ ] ①做好后续维护兑现服务承诺：
- [ ] 争取尽量多的陪看带看并反馈，
- [ ] 及时反馈业务动作，反馈打扫、检查情况
- [ ] 及时与业主沟通，
