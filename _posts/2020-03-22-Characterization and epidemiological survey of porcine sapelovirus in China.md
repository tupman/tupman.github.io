## ABSTRACT

- [ ] 猪脑病毒(PSV)是猪急性腹泻、呼吸窘迫、生殖功能衰竭和脊髓灰质炎的病原体。在此，我们报道了从猪腹泻标本中分离到的PSV的分离、基因组序列和生物学特性。
- [ ] 在我们的研究中，两株PSV的直径约为25nm，其基因组长度为7564个核苷酸。命名为PSV-. JXXY-a2和PSV-JXXY-C. .系统发育分析表明，这两株病毒均属于中国分离株。
- [ ] 此外，PSV-JXXY-a2菌株在54°C时可迅速失活，并可在猪、人和小仓鼠来源的不同细胞系上生长。
- [ ] 致病性调查表明，分离的PSV能有效地感染新生仔猪，引起仔猪腹泻。进-步的流行病学调查表明，猪群中PSV的流行率较高，腹泻猪的PSV阳性率明显高于中国无症状的猪群。我们的研究结果表明，PSV-JXXY-a2对新生仔猪具有致病性，并提高了对PSV感染流行的认识。

## Keywords

**Porcine sapelovirus (PSV)**

## 1. Introduction

- [ ] 猪皂甙病毒(Psv)是一种无包膜、阳性的单股基因组rna病毒，筲在肠道病毒属中被命名为猪肠病毒8型，现被归类为皮氏科Sapelovirus属(Sapelovirus)。到目前为止, Sapelovirus属由猪Sapelovirus、猴Sideelovirus. 离类Sideelovirus和未分类的Sapelovirus(包括蝙蝠、加利福尼亚海狮、土拨鼠、小鼠和WUHARV Sideelovirus)组成。
- [ ] PSV的基因组约为7.5kb，其基因组结构与其他微微病毒相似:一个5、 非翻译区域(UTR),一个大的开放阅读框(ORF)，一个3’UTR和一个聚(A)尾。
- [ ] 大的ORF编码一个单- -的多蛋白，随后被切割成四个结构蛋白(VP 4, VP2, VP3和VP1)和七个功能蛋白(2A，2B, 2C，3A, 3B，3C和3D)。
- [ ] 据报道，它与猪的急性腹泻、呼吸窘迫、生殖功能衰竭和脊髓灰质炎有关，主要通过粪便-口途径传播，尽管猪的PSV感染最常见的是无症状。
- [ ] -项来自韩国的研究显示，腹泻标本与非腹泻标本的PSV感染率没有差异，但是，另-种粪便样本中的病毒亚组学分析表明，Psv在腹泻猪中比在健康猪中更普遍。
- [ ] 值得注意的是，最近的一份报告显示，韩国PSV sv-A株在仔猪中是肠道致病性的，在肠道产生明显的损伤，并伴随着高病毒RNA载量的严重绒毛菱缩。
- [ ] 在此，我们描述了从猪腹泻粪便中分离到的PSV,以及分离的PSV的分子和生物学特性。此外，分子和血清学研究表明，PSV在中国猪中高度流行。

## 2. Materials and methods

### 2.1. Cells

- [ ] 猪睾丸(ST)细胞(由浙江大学黄耀威教授提供)、猪肾(PK-15)细胞(CCL 33; ATCC)、 猪肠_上皮细胞(IPEC-J2)(C 0668;.上海官生.物技术有限公司，中国)、小仓鼠肾(CCL-10).人胚肾(293 T)细胞(CRL 11268;ATCC)、人肺泡基底上皮细胞(CCL-185;ATCC)、马丁-达尔比犬肾(MDCL-34;ATCC)和鸡成纤维细胞(CRL-1746); .ATCC(ATCC)在我们的实验室中被维护。
- [ ] ST、BHK-21、293 T、A 549、MDCK和DF-1细胞在37°CDulbecco改良Eagle 's培养基(DMEM;吉布co)中，在5%CO2中培养。pK-15细胞在37°C、5%CO2条件下，在最低;必需培养基中培养(MEM; HyClone)。
- [ ] IPEC-J2细胞在37°C、5%CO2、RPMI-1 640培养基中培养。
- [ ] 所有培养基均添加10%胎牛血清(FBS;生物工业)。

### 2.2. Virus isolation

**病毒分离**
- [ ] 2016年，从江西省新余市4个不同农场的3日龄腹泻仔猪中采集了10份腹泻粪便标本。
- [ ] 三个样(XXY-A1、-a2和-A3)来自A农场;三个样(本(XY-b1、-b2和-b3)来自8场; 三个样((XY-F3. -F4和-F5)来自C农场: 一个样本(X-)来自农场D.
- [ ] 用ST细胞对10例腹泻粪便标本进行病毒分离。用含抗生素(青霉素和链霉素)的DMEM稀释10倍，30s，3500xg离心10 min.上清液经0.22μm孔滤膜过滤(美国默克公司)。
- [ ] 用过滤上清液接种37°C和5%CO2条件下的ST细胞汇合单层，每日观察3天，观察细胞病变(CPES)的发展。
- [ ] 感染后3d,取培养上清液，传代4次。用菌斑法纯化病毒分离物。

### 2.3. RNA extraction, deep sequencing and assembly

**RNA提取、深度测序和组装**
- [ ] 利用QIAamp病毒RNA迷你试剂盒(QIAGEN,德国)从空斑纯化病毒中提取病毒RNA，并在Illumina MiSeq平台上进行下代测序(Huada基因技术有限公司)。
- [ ] 病毒RNA经AgilentTechnologies 2100生物分析仪鉴定合格。
- [ ] 采用随机引物N6，采用上标Ⅱ逆转录酶(Invitrogen)进行逆转录聚合酶链反应(RT-PCR)。
- [ ] 随后，使用bgi(bgi-深圳)提供的标准协议构建了插入大小为130-170碱基对的配对端库，然后在bgiseq-500平台上以pe 100模式进行了测序。
- [ ] 原始数据由SOAPnuke获得并过滤。在Susscrofa参考基因组的基础上,利用SOAP对高质量的读取数据进行了处理，以去除宿主污染物的读取。非主机读取被映射到病毒数据库，使用SOAP来筛选候选病毒(有映射的读取≥100的数量)。
- [ ] 非主机读取是重新组装使用IDBA-UD的默认选项。我们还使用Maq(Li等人，2008年)对PSV基因组进行了基于参考的组装，并改进了IDBA-UD的结果。
- [ ] 最后，除5‘和3’ UTR序列外，还利用BLAST(Altschul等人，1990年)在NCBI核苷酸数据库中获得和鉴定了PSV基因组的主要片段。
- [ ] 为了获得两株分离的PSV株的全基因组，设计弓|物对韩国PSV株KS 04105进行5‘和3’UTR序列的扩增。PCR产物用TIghtTA克隆试剂盒分别纯化和克隆。然后用Sanger测序来确定序列。
- [ ] 利用DNAStar程序对基因组进行组装和分析。

### 2.4. Preparation of anti-PSV antibody

**抗PSV抗体的制备**
- [ ] 猪抗血清经南京农业大学动物保护与伦理机构委员会批准获得。6头1日龄常规猪分为两组(n=3),分别圈养。
- [ ] 第1组用1.5mL灭活的PSV抗原与佐剂混合免疫3次(间隔10d)。
- [ ] 将6份福尔马M(Sigm)5与4SU-XY-2上清液(106.94 pfu/mnL在室溫下孵育12h,用蓉名的乳化佐剂弗氏完全佐剂(Sgma)进行匀浆，制备灭活病毒。
- [ ] 2组仔猪为阴性对照。最后-次免疫后2周静脉注射Euthol液终止仔猪，取每头仔猪血清，于-80°C保存。

### 2.5. Immunofluorescence assay (IFA)

**免疫荧光法**
- [ ] 为检测PSV对各种细胞的侵染能力，分别用PSV-JXXY-a2、IPEC-J2、 BHK-21、 293T、A 549、MDCK和DF-1细胞接种PSV-JXXY-a2，多重感染(MOI)0.5。
- [ ] 接种ST、PK-15、 BHK-21和293 T细胞10h,接种IPEC-J2、 A 549、MDCK和DF-1细胞48h。细胞孵育后用4%多聚甲醛固定，37°C与抗PSV血清(稀释1: 100)孵育 1h, 37°C用FITC-共轭-二次抗体(KPL)孵育1h.
- [ ] 用10μg/mL dapi(Roche)染色5 min,荧光显微镜观察样品。

### 2.6. In vitro growth characterization and temperature sensitivity

**体外生长特性与温度敏感性**
- [ ] 用菌斑法测定psv-jxy-a2在每个时间点的病毒滴度,并对其进行修饰。
- [ ] 简单地说，覆盖细胞37°C孵育60h后，用1%(w/v)结晶紫在20%Etoh和5%甲醛中固定和染色至少2h.计数斑块，病毒滴度为PFU/ml。
- [ ] 为进一步研究PSV的温 度敏感性，在37、42、50、 51、52、 53、 54和55°C温度 下加热PSVJXXY-a2，用IFA法测定ST细胞中病毒滴度(TCID 50/mL)。

### 2.7. Virus purification and electron microscopy

**病毒纯化与电镜**
- [ ] 用PSV- JXXY-a2感染ST细胞单层，MOI为0.5, 感染48h.细胞经三次冻融循环。用4807xg，4°C离心15 min澄清细胞碎片.
- [ ] 用超离心法从澄清上清液中分离出粗病毒,在4'C下离心92600xg，培养3h。病毒颗粒再悬浮于0.4mL DMEM中，然后在20~ 60%(w/v)的不连续碘乙醇溶液(Optiprep"密度梯度培养基，Sigma-Aldrich)上离心，在4°C下离心2.5 h.
- [ ] 利用日立H-7650型透射电镜(TEM)对该界面的病毒带进行采集和检测。

### 2.8. Sequence and phylogenetic analysis

**序列与系统发育分析**
- [ ] 对PSV分高株的多白基因和个体基因的核苷酸序列和推导的氨基酸序列进行了比较，并与其他已知的微微病毒(充表S2)的核音酸序列和推导的氨基酸序列进行了比较。
- [ ] 在mega 7.0中，基于核苷酸序列，采用邻接法进行了系统发育分析。用1000个复制值计算引导值。pspv vp1蛋白的结构同源性模拟是由瑞士模型服务器进行的。

### 2.9. Animal experiments

**动物实验**
- [ ] 动物实验的伦理认可被描述为猪产性的抗血清。
- [ ] 从一头母猪(PSV)、猪流行性腹泻病毒(PEDV)、猪三角冠状病毒(PDCoV)、传染性胃肠炎病毒(TGEV)、A型轮状病毒(GARV)和哺乳动物原肠病毒(MRV)的粪便标本中筛选出rna阴性、猪流行性腹泻病毒(PEDV)、猪三角冠状病毒(PDCoV)和抗体阴性。
- [ ] 仔猪用母乳替代饲料(中国安友)代替初乳和母乳.4只1日龄新生仔猪随机分为两组.
- [ ] 第1组(1-V2)分别口服PSV-JXXY-a25mL(106.94 pfu/mL, 108.20拷贝/mL);第2组(C1-C2)仔猪口服DMEM 5mL作为阴性对照。
- [ ] 每天监测所有仔猪的直肠温度和临床症状,并采用主观评分法对每个仔猪进行粪便一致性评分，其中0是正常的，1是糊状的，2是奶油的，3是水的。
- [ ] 粪便稠度评分为2分或3分的仔猪腹泻呈阳性。
- [ ] 取仔猪粪便拭子，病毒RNA检测值分别为0、1、2、3和4dpi.所有仔猪静脉注射Euthol溶液，5 dpi进行安乐死。
- [ ] 取每个仔猪的心、肝、脾、肺、肾、扁桃体、小脑、大脑、脊髓、上颌下淋巴结、腹股沟淋巴结、膀胱和消化道进行病毒RNA检测，并分别从胃、十二指肠、空肠、回肠、盲肠、结肠、直肠和肠系膜淋巴结中抽取消化道标本。
- [ ] 从粪便或组织中分离出的病毒采用qRT-PCR方法进行检测，如上文所述。取V1、V2仔猪肠道内容物进行ST细胞病毒再分离。

### 2.10. Epidemiological investigation

**流行病学调查**
- [ ] 为进行PSV分子流行病学调查，在2016年2月至2017年3月期间，对安徽、福建、江西、广西、浙江、江苏、湖北、湖南、山东、辽宁、上海等省不同商品猪群中无症状或临床腹泻的母猪、育雏者和哺乳仔猪共采集了185份临床标本，包括粪便、粪便、粪便和肠道。
- [ ] 对所有标本进行了PSV和猪流行性腹泻病毒(PEDV)的检测。
- [ ] 如Chen等人先前报道的那样，用qRT-PCR筛选5‘非翻译区进行PSV检测。用于检测PEDV的方法和以前报道的一样。
- [ ] 为进行血清学流行病学调查，于2017年2月至12月在甘肃、山西、河北、北京、山东、江苏、安徽、浙江、福建、江西、湖南、湖北、广西和海南等省的商品猪群采集了258份猪血清样本。
- [ ] 如前所述，IFA测定了PSV特异性抗体滴度.
- [ ] 简单地说，用PSV在MOI 1下接种12h,然后用258份4倍系列稀释法的血清样品进行IFA.所有样本一示三份。

### 2.11. Statistical analysis

**统计分析**
- [ ] 计算汇总统计数字是为了评估数据的总体质量。所有数据均使用GraphPad Prism软件(7.0版)进行处理。
- [ ] 体外实验至少独立进行3次，用t检验评价病毒滴度的统计学意义。经统计学处理，P值为0.05.

## 3. Results

### 3.1. Isolation and identification of PSV strains

**PSV菌株的分离与鉴定**
- [ ] 用10份PDCoV RNA阳性但PEDV、TGEV.GARV和MRVRNA阳性的腹泻粪便标本接种ST细胞。在ST细胞单分子膜上进行2次盲传代后，经48-52 hpi后，JXXY-a2和jxyc两种样品均呈现收缩、四舍五入和提升形态的cpe.
- [ ] 出乎意料的是，从这两个分离物中提取的RNA对PDCoV是阴性的。
- [ ] 经4次连续传代，接种后的ST细胞在15-18hpi后出现明显的cpe,与模拟感染的细胞相比，感染的细胞单层被24-36 hpi完全分离。
- [ ] 菌斑分析显示，经三次净化后，斑块直径约为3.5mm.TEM观察显示，二十面体、无包膜的病毒颗粒直径约为25 nm.
- [ ] 下一代测序显示，这两个分离物是PSV,没有发现其他病毒匹配。这些结果证实了这两株病毒均为psv-jxy-a2和psv-jxy-C.

### 3.2. Full genome sequence and phylogenetic analyses

**全基因组序列和系统发育分析**
- [ ] 两个分离物PSV- JXXY-a2和PSV-JXXY-c具有相同的基因组结构，不包括聚(A)尾.
- [ ] 全基因组长度为7564 nt,包括一个489 nt5 'UTR序列， 一个6996 nt多蛋白基因和个79 nt 3' UTR序列。如补充表2所示，PSVJXXY-a2/c的全基因组长度与所有其他参考株不同。
- [ ] PSV菌株的多蛋白基因长度在6969-6999 nt之间;然而，除具有三核苷酸插入的HuN 4株外,在中国有6996 nt长度的PSV中，它是相对保守的。
- [ ] 根据先前的研究，PSV的5端核苷酸残基应该是Uu。PSV-JXXY-a2/c序列与这些序列特征一致,但在参考菌株中，只有KS 04105、KS .05151、KS 055217和IA 33375具有完整的5 'UTR序列;在这些序列之间，5 UTR的长度从489 nt到491 nt.完整的3 'UTR序列已被广 泛报道，其长度相对保守，通常为79 nt和82 nt。
- [ ] 对PSV基因组的编码区，包括多蛋白基因、衣壳编码区(P1基因)和从P1蛋白中切割的成.熟肽(VP1、VP2和VP3)的基因进行了系统遗传学分析(图2)。
- [ ] 多蛋白基因的系统进化树分析表明，该属病毒与肠道病毒的亲缘关系比与Teschovirus的亲缘关系更近。在猪脑病毒属中，猪脑病毒与蝙蝠脑病毒的关系最为密切。
- [ ] PSV分离株连同ncbi数据库中提供的25个完整的picornavirus毒株的多蛋白序列显示，所有PSV毒株都被划分为与地理位置相关的三个不同的聚类，即中国、韩国和美国/英国/印度。
- [ ] PSV-JXXY-a2和PSV-JXXY-c属于中国聚类,与韩国聚类关系更为密切。

### 3.3. Comparative analysis of the deduced amino acid sequence of the polyprotein

**多聚蛋白氨基酸序列的比较分析**
- [ ] 为探讨PSVS与PSV-JXXY-a2和PSV-JXXY-C的遗传关系和特性，将所推导的多聚蛋白氨基酸序列与聚蛋白基因系统发生树中不同类群的典型参考株进行了序列比对。
- [ ] 结果表明，PSV-JXXY-a2和PSV-JXXYc蛋白的氨基酸序列是一致的， 与中国株HuN2/3/4有许多相似之处。
- [ ] 与其他参考菌株相比，PSV-JXXY-a2/C在12种成熟肽(-VP4-VP2-VP3-VP1-2A-2B-2C-3A-3B-3Cpro-3Dpro)中均有氨基酸贊换，其中VP2、VP3. VP1. 2C、 3A和3C均有明显的替换。
- [ ] 在这些成熟肽中，VP1是高变蛋白，有三个突变簇，3 '端序列多变。突变簇1，包括4个相邻的氨基酸突变,位于750-752和754个位置。
- [ ] 突变体2，包括5个氨基酸突变，位于771,772和775-777位置。
- [ ] 突变簇3，包括6个氨基酸突变，分别位于844、845和847-850位点。
- [ ] PSV VP1单体蛋白与人肠道病毒71 VP1蛋白模板(瑞士模板库中的3zfe.1.A)的结构同源性表明，这三个VP1蛋白突变簇位于病毒表面的三个环中(图3B)。

### 3.4. Temperature sensitivity and cellular adaptation of the PSV isolate

**PSV分离物的温度敏感性和细胞适应性**
- [ ] 为检测病毒的生长动力学，将PSV-JXXY-a2接种于MOI为0.5的时间点。病毒滴度呈逐渐上升的趋势,在18 hPl(图4A)时达到最高(107.57 pfu/mL),说明PSV-JXXVa2株能有效地在ST细胞内感染和复制。
- [ ] 为检测温度敏感性，PSV菌株|XXY-a2在规定温度下加热1h,接种ST细胞单层膜。.
- [ ] 与未处理对照组相比，37°C染毒1h后，JXXY-a2的传染性显著降低，且随着处理温度的持续升高，病毒滴度继续下降(图4B)。
- [ ] JXXY-a2在53°C时保持感染力1 h,在54°C下完全灭活6 min(图4C)。
- [ ] 结果表明，54°C是最有效的灭活温度。
- [ ] 为评价PSV株JXXY-a2对不同细胞株的适应能力，以ST、PK-15、 IPEC-J2、 BHK-21.293 T、A 549、 MDCK和DF- 1细胞为材料,检测了PSV株JXXY-a2的感染能力。
- [ ] 如图4D所示，JXXY-a2不仅能感染ST、PK-15和IPEC-J2，而且能很有效地感染BHK-21和293 T.
- [ ] A 549和MDCK对PSV感染不敏感。
- [ ] 结果表明，PSV-JXXY-a2菌株在体外具有广泛的细胞定向性。

### 3.5. PSV is pathogenic to neonatal piglets

**PSV对新生仔猪的致病性**
- [ ] 为确定分离的PSV-JXXY-a2株是否为猪肠道致病菌两只1日龄、新生、健康的常规仔猪接种PSV-JXXY-a2.
- [ ] 4只仔猪在1.2. 3 dpi时均未观察到明显的温度变化或明显的临床症状。在4dpi时，两只接种PSV的仔猪出现水样腹泻(3分)，而两只模拟接种仔猪(图5A-B)。
- [ ] 由于两只接种PSV的仔猪出现严重的腹泻症状，所有仔猪的终止时间均在5 dpi.
- [ ] 为了进-步探讨PSV口服液中病毒的脱落和病毒抗原的分布，每天采集仔猪粪便拭子和各仔猪组织标本进行病毒RNA检测。粪便病毒脱落显示，在2-4 dpi时，pSV感染的猪粪便中会产生高水平的病毒RNA(图5C)。
- [ ] 此外，在所有的消化道器官都能检测到病毒RNA，盲肠、结肠和直肠的病毒RNA水平高于十二指肠、空肠、回肠、胃和肠系膜淋巴结(图5D)。
- [ ] 此外，在扁桃体、腹股沟淋巴结和膀胱中,PSV RNA水平高于脾脏、肺、小脑、大脑和上颌下淋巴结，而在心、肝、肾和脊髓中未检测到病毒RNA(图5e)。
- [ ] 此外，从PSV感染的肠内容物中分离出的病毒在42 HPI时表现出明显的CPE，在IFA中观察到大量的PSV阳性细胞，其抗血清特异h性为PSV(图5A)。
- [ ] 这些数据表明，PSV-JXXY-a2对仔猪具有致病性。

### 3.6. Epidemiological investigation of PSV infection in pig herds

**猪群PSV感染的流行病学调查**
- [ ] 为了评估PSV感染的频率，我们收集了185份标本，其中临床腹泻138份,无症状47份。如图6B所示，在185份标本中，75份(40.54%), 72份(39.92%), 32分(17.30%)为PSV/PEDV双阳性。

- [ ] 同时，采用IFA对258头猪进行血清流行病学调查。如图6d所示，在采集的血清中，PSV阳性率.较高(80.23%，207/258).68.99%(178/258)的样本滴度大于43,半数以上的滴度在43~44之间，10.85%(18/258)的滴度大于45.

## 4. Discussion

- [ ] 总之，我们从201 6年江西省猪场腹泻标本中分离出PSV-JXXY-a2和PSV-JXXY-c两株PSV,并对其进行了序列分析和遗传鉴定，并对新生仔猪的致病性进行了研究。
- [ ] PSV衣壳蛋白vp1是一种最外部的病毒蛋白，含有许多主要的中和位点。VP1蛋白被认为是免疫显性的，在确定微微病毒之间的遗传关系方面具有重要价值。
- [ ] 对推导的VP1氨基酸序列进行同源性比较发现，PSVJXXY-a2、PSV-JXXY-c和中国匈奴毒株与其他参考毒株相比有3个突变簇
- [ ] 结构同源性模拟发现，这三个突变簇位于VP1蛋白的表面环上。病毒表面暴露的环是微微病毒病毒的最易变化的区域，被认为是最重要的中和免疫原位点。
- [ ] 因此，这三个突变簇是否与中和免疫原性有关，还有待进一步研究。
- [ ] 此外，VP1的终止也能诱导许多微微病毒的中和抗体。因此，PSV VP1蛋白3‘端序列的变化是否与中和抗原位点有关，这是值得注意的。
- [ ] 由于PSV-JXXY-a2和PSV-JXXY-c的核苷酸序列和推导的氨基酸序列相近，用代表菌株PSV-JXXY-a2对PSV进行了体外生长特性、温度敏感性测定和体内致病性试验。
- [ ] 此前的一项研究报道，PSV感染仅限于猪源性细胞。
- [ ] 另一项研究表明，人类细胞株PLC/PRF/5.HepG 2/C3a和绿色猴子细胞系Vero E6和PGMKC对PSV感染也很敏感。
- [ ] 我们的研究表明，除猪细胞株ST、PK-15和IPEC-J2外，人293 T细胞株和小仓鼠BHK-21细胞株均对PSV感染敏感。总之，人绿猴和小仓鼠细胞感染PSV引起了人们对其潜在宿主范围超出猪的关注。
- [ ] 但人A 549细胞株、犬MDCK细胞株和鸡成纤维细胞株DF- 1对PSV均有耐药性,对PSV敏感性差异的研究有助于PSV受体的研究。
- [ ] 虽然PSV分布广泛，但在65°C加热5 min或60°C加热10 min后，其灭活效果较好。结果表明，灭活PSV-JXXY-a2的最低条件是在54°C下加热6 min,这可能是由于PSV菌株的不同所致。
- [ ] 到目前为止，对PSV的致病性研究较少，只有一株韩国SV-A具有良好的肠道致病性特征;它能引起仔猪腹泻和肠道病变。
- [ ] 本研究发现腹泻粪源PSV株PSV-JXXY-a2能.在肠道内有效复制并诱发仔猪腹泻，提示该.株PSV-JXXYa2可能是致病性的。
- [ ] 值得注意的是，本研究的流行病学调查表明，腹泻样品中PSV/PEDV的阳性率和共感染率明显高于无症状标本，表明PSV可能与.猪肠道疾病及其它猪病原体共同作用;因此，在控制猪腹泻时，不能忽视PSV.
