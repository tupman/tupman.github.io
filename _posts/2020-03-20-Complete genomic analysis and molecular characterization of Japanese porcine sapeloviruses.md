## Abstract

- [ ] 猪Sapelovirus(PSV是-种可引起多种疾病的猪肠道病毒。然而，很少有报道描述PSV基因组的分子特征。本研究利用生物信息学技术对日本猪检测到的23个PSV基因组进行了分析。
- [ ] 对顺式活性RNA元件的分析表明，预测的5‘非翻译区(UTR)内核糖体进入位点的二级结构和2C编码区的顺式复制元件在PSVS中是保守的。相反，不同的PSV在3 'UTR 上的值是不同的。然而，结构域间的三级结构在所有PSV中都是保守的。
- [ ] VP1区核苷酸序列的系统发育分析表明，PSVS具有序列多样性;然而，由于簇的自举支持度低，它们不能被归类为基因型。
- [ ] C端VP1区的插入和/或缺失模式，与VP1树的拓扑结构无关。3CD系统发生树与VP1树在拓扑上不同，同一国家的PSV是独立聚类的。
- [ ] 重组分析表明，P2区上游存在重组事件，部分重组断点与C端VP1区的插入和/或缺失有关。
- [ ] 这些发现表明，PSVS表现出遗传多样性和频繁的重组事件，特别是在P2区域的上游; 然而，PSVS目前还不能被归类为基因型，并且在所有PSVS中都能观察到顺式活性RNA的保守遗传结构特征。

## Introduction

- [ ] Sapelovirus属毕赤病毒科，包括> 30属和>75种，其中包括Sapelovirus A和Sapelovirus B,以前分别被称为猪Sapelovirus(PSV)和Simian Sapelovirus。
- [ ] 该病毒目前被命名为离流感病毒，属于Sapelovirus属。最初，PSV被指定为猪肠病毒(PEV)8。在对PEV进行了进一步的基因组研究后， PEV1-7和PEV 11-13被重新分类为Teschovirus属;PEV 9和10被正式纳入PEV-B,改名为EV-G，PEV 8正式包括在PEV-A中，被重新命名为PSV。
- [ ] PSV可弓|起各种症状的疾病，包括脊髓灰质炎、腹泻、肺炎和生殖障碍。然而，在健康猪的粪便中也发现了psv。在中国、韩国、美国、欧洲国家、印度、巴西和日本的猪体内都发现了PSV.
- [ ] psv病毒是通过粪口传播途径在猪群中传播的，被认为是猪体内普遍存在的肠道病毒。
- [ ] psV有一个大约7.5 k长的单链ma基因组，其正向极性包合一个大的开放阅读框 (ORF)。ORF编码5 '和3’ 非翻译区(UTRs)两侧的单个多蛋白和3 'UTR末端的PolyA尾。单个多蛋白随后被加工成一个先导蛋白，4个结构蛋白(VP4，VP2, VP3和VP1)来自P1,7个非结构蛋白(2A，2B和2C来自P2，3A,3B,3C和3d来自P3)。
- [ ] 5 'UTR和3' UTR已知含有重要的结构单元。一个位于5 'UTR中的内部核糖体进入位点(IRES)对于启动从无上限病毒基因组中的翻译是必要的。
- [ ] 目前已知的IRES有5种类型，根据其核苷酸序列和结构特征进行分类，并对其翻译起始机制进行了初步的研究。Picornavirus的3 'UTR含有 =级和三级结构元素，是有效进行病毒RNA复制所必需的。
- [ ] 除了5 'UTR和3’ UTR结构元件外，在多个picornavirus基因组的编码区中还发现了有效启动病毒互补RNA链所需的顺式作用复制元件(Cres)。.
- [ ] 肠道病毒G、whi、b、b、ngstheEv/Sapelovirus超群与PSV共20种基因型。这些基因型在vp1基因区域存在25%以上的核苷酸序列差异。
- [ ] 另-方面，虽然有关于PSV抗原多样性的报道，但还没有关于PSV的血清型或基因型的报道。
- [ ] 本研究利用DDBJ/EMBL/GenBank数据库获得的PSV序列，对日本猪粪便中检测到的23个PSV基因组进行了生物信息学和系统遗传学分析，并与其他国家的PSV进行了比较。

## Materials and methods

### 1.Metagenomic and whole genome analysis of PSV

**PSV的亚基因组和全基因组分析**
- [ ] 在2014至201 6年对222个腹泻猪粪便样本进行的一项关于肠道病毒G型的日本研究中，共鉴定出21个序列连体，其长度为> 7000个碱基，与PSV序列同源。另外两个> 7000碱基PSV序列连体是从2017年至201 8年在日本从健康猪身上采集的另外8个粪便样本中发现的，使用的是元组学方法。
- [ ] 从10%粪悬液上清液中直接提取总RNA。
- [ ] 从总RNA中提取cDNA文库，用NEBNext超RNA库Ilumina准备试剂盒(NewEnglandBiolabs，Ipswich, MA, USA)制备cDNA文库。
- [ ] 采用MiSeq台式测序仪(llumina，SanDiego, CA, USA)进行深度测序。MiSeq Reporter(llumina)创建的FASTQ文件被导入CLC基因组工作台7.5.5(CLC BIO,丹麦奥胡斯)。
- [ ] Contigs是通过使用denovo程序集裁剪读取创建的,在CLC基因组I作台中使用了默认参数。
- [ ] 为了评估生成的连续体的置信度，利用CLC基因组学工作台对一个引用进行映射读取，其中参数设置最严格(失配成本2;插入成本3;删除成本3;长度函数0.9;相似函数0.9),省略了读取深度不足的5 '和3’ 序列(<3)。
- [ ] 为了获得基因组3‘端的完整序列， 采用商用试剂盒(3’-全RACE核心集， Takara Bio,Otsu, Japan)进行cDNA末端(RACE)的快速扩增。

### 2.Genome analyses

**基因组分析**
- [ ] 所获得的长度超过7000 nts的连续体具有足够的读取覆盖率,并使用集群W对齐。
- [ ] 病毒RNA二级结构的预测是用mfoly程序进行的。
- [ ] 对这些树进行了自举分析，得到了1000个副本。用CLC基因组学工作台7.5.5(CLC BIO)计算成对序列的同源性。
- [ ] 重组分析使用Simplot软件v.3.5.1和重组检测程序(RDP)v.4.80进行。

## Results

### 1.Identification of nearly complete genome of PSVs

**PSVS几乎完全基因组的鉴定**
- [ ] 采用元组学方法，从10日龄到80日龄猪共鉴定出长度为7000个碱基的23个Contigs。
- [ ] 经BLAST分析证实，这些Contigs均为萨佩洛病毒序列。在10~80日龄猪腹泻粪便标本中检出5个序列，其中5个序列与猪流行性腹泻病毒(1个样本)、Picobirnavirus(3个样品)、Aichivirus C(3个样本)、Teschovirus(2个样本)、EV-G(1个样本)和Rotavirus A(1个样本)的序列相似。
- [ ] 在15~60日龄无腹泻猪粪便标本中发现18个序例，其序列与Teschovirus(1份)、 猪星形病毒(14份)、轮状病毒C(4份). 肠病毒G(5份)、Picobirnavirus(6份)、 sapovirus(5份)、猪病毒(1份)和后病毒(1份)的序列相似。表1概述了在本研究过程中获得的有关日本PSV株的信息。

### 2.Analysis of the secondary RNA structure within the 5′ UTR, 2C coding region, and 3′ UTR

**5 'UTR、 2C编码区和3’ UTR的二级RNA结构分析**
- [ ] 用Mfoly程序对5 'UTR. 2C编码区的CRE和3' UTR的二级RNA结构进行了预测。CRE位于PSV基因组中2C的编码区,Cres含有一个必需的AAACA基序。
- [ ] 具有AAACA基序的茎环结构(对应于PSV13/1957/GBR的基因组位置4419-4478)被保存;然而，PSV之间的差异很小(图1)。
- [ ] 所有发夹环包含16个NTS;然而，42个PSV(包括16个日本PSV)包含一个AAACA基序，而29个PSV(包括7个日本PSV)有2个AAACA基序(图1)。
- [ ] 为了获得完整的3 'UTR序列, 采用RACE方法，确定了8个菌株(Ishi-Ya8/2015、HgTa2-1、HgOg 11/2018、Mol2-2/2015、Ishi-IM3/2015. Ishi-Miya3/2015、HkKa2-2/2015和DeTk2-2/201 5)的完整序列。
- [ ] PSV的3 'UTR为82-86NTS长， 不包括聚-A 尾（ poly-A tail）。
- [ ] 最长的3 UTR出现在HkKa2 2/2015和DeTk2-2/2015(图2A)。二级结构分析表明，PSVS具有3个(结构域x Y和Z)或4个(结构域X Y-1、Y-2和Z)的茎环元件，可分为10种类型。.
- [ ] 日本PSV分为七种模式。Ishi-IM3、 IshiMiya3/2015和DeTk2-2/2015除具有共同结构域X、Y和Z外，还具有小茎环结构域Y-2。
- [ ] 虽然PSVS的二级结构不同，但X和z结构域之间的三级相互作用是保守的。在Jpsv 1315/2009/JPN的3 'UTR中， X和Z结构域的环之间发生了7个NT相互作用(登录号: )。LC 326555), Jpsv447/2009/JPN(加入号。LC 326556)和V13/1957/GBR;相反，其他PSV的3 'UTR表现出8个NTS的相互作用，发生在两个组成发夹的环之间(图2B)。
- [ ] 在NT169-443位点上的PSV V13/1957/GBR序列，对DDBJ/EMBL/GenBank数据库中日本PSV和PSV 5 'UTR中的IRES茎环结构域Ⅱ和Ⅲ进行了比对和比较。
- [ ] 这些区域的序列，特别是在区域IID、lie和Ilif中的序列是高度保守的。
- [ ] 这些区域的二级结构分析表明，在所有psv中，区域 II, IIIa, IIIb, IIId, IIIe, 和 IIIf的茎环结构都是相同的，并且它们中的任何-一个都不存在结构域ⅢC。

### 3.Phylogenetic tree analysis, pairwise sequence comparison, similarity plot analysis,and recombination analysis

**系统发育树分析、配对序列此较、相似图分析和重组分析**
- [ ] 利用DDBJ/EMBL/GenBank数据库中23个日本PSV的VP1区和52个PSV株的NT序列，用ML方法构建系统发育树。
- [ ] 日本PSV的分支至少有6个系，其中只有来自中国和美国等其他国家的日本株或PSV。PSV表现出遗传多样性，形成多个聚类，但有些聚类不受≥70%引导值的支持(图3A)。
- [ ] VP1编码区的3 '端存在插入和/或缺失， 且有不同的模式，但插入和/或缺失的模式与VP1树的拓扑结构无关(图3B)。
- [ ] 利用DDBJ/EMBL/GenBank数据库中日本PSVS和PSVS的3CDNT序列构建的系统发生树表明，该树的拓扑结构与VP1树的拓扑结构不一致。有趣的是，PSV形成了集群，它们对应于每个PSV被隔离的国家
- [ ] 仅日本PSV就形成了两个血统(图3C)。用CLC基因组学I作台进行成对序列计算。结果表明，所有日本PSVS的3CDNTS和氨基酸序列(nt: 88.8~ 100%，AA: 97.2~100%)均具有较高的序列同源性，但与其他国家的PSV同源性分别为86.5%~92.39%和95.8% ~99.7%(附表1、表2)。
- [ ] 中国PSVS在3CD区的同源性(NT:91.0~100%，AA: 98.3-100%)也高于其他国家(NT: 86.8~92.3%， AA:95.7~99.7%)。.
- [ ] 在VP1区，中国株与国外菌株的≥同源性分别为70.8%(Nt)和75.1%(AA)，而中国株分别为≥71.8%和≥76.1%，AA序列同源性为75.1%。最低的NT序列标识为VP1(HuN9/201 6/CHNVS)的70.8%。
- [ ] 由于VP1树和3CD树之间存在不同的拓扑结构，因此进行了重组分析。利用Simplot分析，确定了几个交叉位点。Ishi-Ka2/2015与HgYa2-4/2015在5 'UTR、 L和VP4编码区、P2和P3编码区以及3’UTR上具有较高的nt序列相似性,而VP2、VP3和VP1编码区与Ishi-Ka2/2017高度相似，表明存在重组事件(图4a，b)。
- [ ] HgYa-1/2016在5 'UTR、 P1区和3’半ORF上与HgTa2-2具有显著的序列同源性，而HgYa-1/2016在2A编码区与Mol 3/201 6的序列相似性高于HgTa2-2/2015，表明发生了重组事件(图4A，C)。
- [ ] 几乎所有的交叉位点都被定位在5 'UTR到2B区域。.扫描分析表明，在5 'UTR、 VP4、VP2、VP1、 2A和2B区域发现了可能的重组断点(图4B，C;补充图3)。其中两个可能的重组断点位于VP1编码区
3 '端的nt插入和/或缺失位点(图4B，c)。

## Discussion

- [ ] 利用元组学方法，从日本从10到80日龄猪的230份粪便样本中鉴定出了23个几乎完全的PSV基因组。
- [ ] PSV被认为会导致腹泻，而且PSV经常出现在表面健康的猪身上。
- [ ] 这项研究中，对PSV进行了进一步的分析。
- [ ] 本研究对40个PSV的IRES茎环结构域的5 'UTR序列进行 了序列比较和二级结构分析，共检测到23个日本PSV,这些序列和结构特征在所有PSV中都是高度保守的。
- [ ] 对23个日本PSV的2C区和48个其他国家的PSV的M折叠分析表明，虽然PSV的二级结构略有不同，但茎环结构是很好的保守结构。
- [ ] 在2C区中，发现了一个AAACA基序和两个AAACA基序。
- [ ] cre是一 种重要的元件,它通过vpg尿苷化来合成picornavirus中的正感rna,并为将尿苷添加到vpg中提供了一个模板。
- [ ] 然而，AAACA基序拷贝数的差异的影响还有待阐明。.
- [ ] 与IRES和CRE相比，我们发现了3 'UTRs的各种预测二级结构，可分为10种类型。
- [ ] 3 UTR的结构域X和Y之间的”接吻”相互作用被认为在病毒负链RNA合成中起着重要作用。
- [ ] 虽然在结构域X和Z的循环中观察到7或8个NTS的相互作用，但这种三级结构在所有PSVS中都得到了很好的维持。
- [ ] 这些数据表明，PSVS的基因组RNA具有不同的nt序列。
- [ ] 然而，顺式活性rna元件的二级或三级结构在其基因组rna中，即5 'UTR、 2C和3' UTR中都是保守的。

- [ ] 利用VP1编码区进行了系统发育分析，发现日本PSV具有遗传多样性，它们要么单独与日本PSVS聚类，要么与中美PSVS聚类。
- [ ] 在VP1树中发现了几个簇，但是，许多集群不支持足够的引导值。
- [ ] 我们还利用完全P1区的nt序列构建了-棵系统发育树(补充图4)。
- [ ] P1树的拓扑结构和系统发生聚类与文献中观察到的VP1树和P1树的拓扑结构和系统发生聚类略有不同，说明PSVS目前还不能归类为基因型。
- [ ] VP1是引起宿主免疫的最外层的病毒蛋白之一，VP1序列比较已被用于几种Picornavirus的基因分型。
- [ ] 对于肠道病毒G,属于Picomavirus超级群3和PSV,基因型分类的标准之一是在分离株之间VP1区的25%nt序列差异。
- [ ] PSVS的VP1 nt和AA序列同源性分别为> 70.8%和> 75.1%，与猪肠道病毒的VP1 nt和AA序列同源性(分别为53.9%和53.3%)相比差异不大。
- [ ] 这可能是人们所知道的psp基因分型困难的原因之一。
- [ ] 进一步的研究包括在PSV VP1区积累了更多的序列数据后系统发生树的重建，以及用抗血清或从血清阳性猪中分离的血清进行抗原性试验来精确地分类PSV.

- [ ] 重组分析结果表明，日本PSVS与其可能的重组断点之间可能发生的rna重组事件的位点位于2B区域的上游，这与先前进行的研究结果一致。
- [ ] 一些可能的重组断点位于C端VP1区域的nt插入和/或删除位置。
- [ ] 然而，插入和/或缺失模式与VP1系统发育树的拓扑结构无关，说明RNA重组事件不属于VP1系统发育的范畴。
- [ ] 3CD系统发育树的拓扑结构不同于VP1树，每个国家的PSV菌株被聚在一起。
- [ ] 这表明VP1和3CD区域的进化是独立的,3CD区域可能较少受到RNA重组事件的影响，因为在此和先前的研究中发现的可能的重组断点位于2B区域的上游。

- [ ] 总之，本研究通过系统发育分析和成对序列比较得出的结果表明，psv基因是不同的。
- [ ] 然而，目前还不能将PSVS划分为基因型。
- [ ] RNA重组事件发生在2B区上游，其中一些与C端VP1区的插入和/或缺失有关。
- [ ] 遗传结构特征在所有PSVS中都是保守的，这对于病毒基因组的复制是很重要的。
- [ ] 这些研究结果可为进一步评估PSVS的流行病学基础和揭示PSVS的进化机制提供依据。
